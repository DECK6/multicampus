리스트로 데이터프레임 생성
ChatGPT: Create a pandas DataFrame from list example
import pandas as pd

data = [
  ['김파이', 25, '개발자'],
  ['이판다', 30, '분석가'],
  ['박데이터', 28, '디자이너']
]

df = pd.DataFrame(data,
  columns=['이름', '나이', '직업'])
  이름   나이   직업
0 김파이  25   개발자
1 이판다  30   분석가
2 박데이터 28   디자이너
