자동화 예제: 실용적인 워크플로우

이번 페이지에서는 지금까지 배운 PyAutoGUI 기능들을 조합하여 실제 작업을 자동화하는 예제를 살펴보겠습니다. 마우스, 키보드, 스크린샷, 윈도우 제어 기능을 함께 사용하여 완전한 자동화 워크플로우를 구현하는 방법을 알아봅니다.

예제: 주기적인 데이터 업데이트 자동화

매일 특정 시간에 엑셀 파일을 열고, 데이터를 새로 고침한 후, 결과를 저장하고 닫는 작업을 자동화하는 스크립트입니다.

1
프로그램 실행 및 창 제어

Excel 프로그램을 시작하고 창을 제어합니다.

import pyautogui
import subprocess
import time

# Excel 실행
subprocess.Popen('excel "report.xlsx"')

# 프로그램이 로드될 때까지 대기
time.sleep(3)

# Excel 창 찾기
excel_window = pyautogui.getWindowsWithTitle('report.xlsx')[0]
excel_window.activate()
excel_window.maximize()
2
데이터 새로고침

단축키를 사용하여 데이터를 새로고침합니다.

# 데이터 탭으로 이동 (Alt+A)
pyautogui.hotkey('alt', 'a')
time.sleep(0.5)

# 새로고침 옵션 선택 (키보드 방향키와 Enter 사용)
# 메뉴 구조에 따라 달라질 수 있음
pyautogui.press('r')
pyautogui.press('a')  # 모두 새로고침 선택
time.sleep(2)  # 새로고침 완료 대기
3
저장 및 종료

파일을 저장하고 프로그램을 종료합니다.

# 저장 (Ctrl+S)
pyautogui.hotkey('ctrl', 's')
time.sleep(1)

# 엑셀 종료
pyautogui.hotkey('alt', 'f4')

# 변경사항 저장 확인 다이얼로그가 나타날 경우
# 저장(S) 버튼 클릭 또는 Enter 키 사용
save_btn = pyautogui.locateOnScreen('save_button.png')
if save_btn:
    pyautogui.click(save_btn)
else:
    pyautogui.press('enter')

print("데이터 업데이트 완료!")
4
예외 처리 및 안전장치

자동화 스크립트의 안정성을 높이는 예외 처리를 추가합니다.

try:
    # 앞서 작성한 자동화 코드
    # ...
except Exception as e:
    print(f"오류 발생: {e}")
    # 스크린샷 찍어 오류 상태 저장
    pyautogui.screenshot('error_state.png')
finally:
    # 실행 중인 Excel이 남아있는지 확인하고 종료
    excel_windows = pyautogui.getWindowsWithTitle('Excel')
    for window in excel_windows:
        window.close()
자동화 스크립트 작성 팁
  • 충분한 대기 시간: 프로그램 로딩, 화면 전환, 다이얼로그 표시 등에 시간 지연을 추가하세요.
  • 이미지 인식 대안: 이미지 인식이 실패할 경우를 대비해 키보드 단축키나 좌표 기반 클릭 등의 대안을 준비하세요.
  • 스크립트 테스트: 자동화 스크립트를 여러 번 테스트하여 다양한 상황에서도 안정적으로 작동하는지 확인하세요.
  • 주기적 실행: Windows의 작업 스케줄러나 cron과 같은 도구를 사용하여 스크립트를 정기적으로 실행할 수 있습니다.
