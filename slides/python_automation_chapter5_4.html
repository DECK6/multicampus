스크린샷 & 이미지 인식

PyAutoGUI를 사용하면 화면 전체 또는 일부분의 스크린샷을 찍고, 이미지를 검색하여 화면에서 특정 요소를 찾아낼 수 있습니다. 이를 통해 화면에 나타나는 버튼, 아이콘, 텍스트 등을 인식하여 자동화 작업을 수행할 수 있습니다.

Pillow(PIL) 필요 Linux에서는 scrot 필요 OpenCV (선택사항, 이미지 매칭 정확도 향상)

스크린샷 캡처

화면 전체 또는 특정 영역의 스크린샷을 찍어 Pillow 이미지 객체로 반환하거나 파일로 저장합니다.

import pyautogui

# 전체 화면 스크린샷 캡처
screenshot = pyautogui.screenshot()

# 스크린샷을 파일로 저장
pyautogui.screenshot('screenshot.png')

# 특정 영역만 캡처 (left, top, width, height)
region_screenshot = pyautogui.screenshot(
    region=(0, 0, 300, 400)
)

화면에서 이미지 찾기

화면에서 특정 이미지의 위치를 찾아 좌표를 반환합니다. 버튼, 아이콘 등을 인식할 때 유용합니다.

# 화면에서 이미지 위치 찾기 (x, y, width, height)
location = pyautogui.locateOnScreen('button.png')

# 여러 일치 항목 모두 찾기 (제너레이터 반환)
locations = pyautogui.locateAllOnScreen('icon.png')
for loc in locations:
    print(loc)  # 각 위치 출력

# 이미지 중앙 좌표 찾기 (클릭하기 좋음)
x, y = pyautogui.locateCenterOnScreen('button.png')
pyautogui.click(x, y)  # 찾은 버튼 클릭

# 간단하게 이미지 클릭
pyautogui.click('button.png')  # 바로 이미지 클릭

이미지 검색 옵션

이미지 검색의 정확도와 속도를 조절할 수 있는 다양한 옵션을 제공합니다.

# 그레이스케일 모드로 검색 (약 30% 속도 향상)
location = pyautogui.locateOnScreen(
    'button.png', grayscale=True
)

# 정확도 조절 (OpenCV 필요)
location = pyautogui.locateOnScreen(
    'button.png', confidence=0.8  # 80% 일치
)

# 특정 영역만 검색하여 속도 향상
location = pyautogui.locateOnScreen(
    'button.png', region=(0, 0, 600, 400)
)

픽셀 색상 확인

화면의 특정 위치 픽셀 색상을 RGB 값으로 확인하고 특정 색상과 일치하는지 검사할 수 있습니다.

# 특정 좌표의 픽셀 색상 가져오기
pixel_color = pyautogui.pixel(100, 200)
# 반환값: RGB(red=130, green=135, blue=144)

# 픽셀 색상이 특정 RGB 값과 일치하는지 확인
if pyautogui.pixelMatchesColor(100, 200, (130, 135, 144)):
    print("색상이 일치합니다!")

# 허용 오차 설정
pyautogui.pixelMatchesColor(100, 200, (140, 125, 134), 
    tolerance=10)  # RGB 각 값 ±10 범위 내 일치
주의사항

이미지 인식 함수는 처리 시간이 길 수 있습니다(1~2초 소요). 화면 해상도, 색상 설정, 스케일링에 따라 인식률이 달라질 수 있으며, 이미지가 찾아지지 않으면 ImageNotFoundException 예외가 발생합니다. 예외 처리와 타임아웃 설정을 고려하세요.
