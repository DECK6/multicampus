ChatGPT 프롬프트 예시
파이썬으로 숫자를 입력받아 1부터 해당 숫자까지의 합을 계산하는 함수를 만들어줘.
위 코드를 더 효율적으로 개선할 수 있을까?
이 함수에 입력값 검증 기능을 추가해줘. 음수나 실수는 처리하지 않도록.
생성된 파이썬 코드
# ChatGPT 프롬프트: 파이썬으로 숫자를 입력받아 1부터 해당 숫자까지의 합을 계산하는 함수를 만들어줘. def calculate_sum(n): # 1부터 n까지의 합을 계산하는 함수 total = 0 for i in range(1, n + 1): total += i return total # 사용자로부터 입력 받기 number = int(input("숫자를 입력하세요: ")) result = calculate_sum(number) print(f"1부터 {number}까지의 합: {result}") # 개선된 버전 - 수학 공식 사용 def calculate_sum_improved(n): # 가우스 공식 사용: sum = n(n+1)/2 return n * (n + 1) // 2 # 입력 검증 추가 def calculate_sum_with_validation(n): # 입력값 검증 if not isinstance(n, int): raise TypeError("정수만 입력 가능합니다.") if n < 0: raise ValueError("음수는 처리할 수 없습니다.") return n * (n + 1) // 2
