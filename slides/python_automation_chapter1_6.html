주요 라이브러리

Pillow (PIL)
이미지 처리 라이브러리로, 이미지 생성, 편집, 텍스트 삽입, 필터 적용 등 다양한 이미지 작업을 수행할 수 있습니다.
ReportLab
PDF 생성 및 편집을 위한 라이브러리로, 전문적인 문서와 보고서를 프로그래밍 방식으로 생성할 수 있습니다.
PyPDF2
PDF 파일을 조작하기 위한 라이브러리로, PDF 병합, 분할, 페이지 추출 등의 작업을 수행할 수 있습니다.
FPDF
간단한 PDF 생성 라이브러리로, 텍스트, 이미지, 표 등을 포함한 PDF 문서를 쉽게 생성할 수 있습니다.

# 수료증 자동 생성 예제 from PIL import Image, ImageDraw, ImageFont import pandas as pd import os from datetime import datetime # 엑셀에서 수료자 데이터 읽기 df = pd.read_excel('수료자_명단.xlsx') # 수료증 템플릿 이미지 열기 template = Image.open('수료증_템플릿.png') draw = ImageDraw.Draw(template) # 폰트 설정 name_font = ImageFont.truetype('font/NanumGothic-Bold.ttf', 60) content_font = ImageFont.truetype('font/NanumGothic.ttf', 30) date_font = ImageFont.truetype('font/NanumGothic.ttf', 25) # 현재 날짜 today = datetime.now().strftime("%Y년 %m월 %d일") # 저장 폴더 생성 if not os.path.exists('생성된_수료증'): os.makedirs('생성된_수료증') # 각 수료자별로 수료증 생성 for index, row in df.iterrows(): # 템플릿 복사본 생성 certificate = template.copy() draw = ImageDraw.Draw(certificate) # 이름 삽입 name = row['이름'] name_width = draw.textlength(name, name_font) draw.text( ((certificate.width - name_width) / 2, 350), name, fill='black', font=name_font ) # 과정명 삽입 course = row['과정명'] course_width = draw.textlength(course, content_font) draw.text( ((certificate.width - course_width) / 2, 450), course, fill='black', font=content_font ) # 날짜 삽입 draw.text( (500, 550), today, fill='black', font=date_font ) # 파일 저장 file_name = f'생성된_수료증/{name}_수료증.png' certificate.save(file_name) print(f"{name}님의 수료증이 생성되었습니다.") print("모든 수료증 생성이 완료되었습니다.")
