자료구조(리스트) 개요
리스트란?

리스트(List)는 파이썬의 기본 자료구조로, 순서가 있는 객체의 집합입니다. 다른 프로그래밍 언어의 배열(Array)과 유사하지만, 더 많은 기능을 제공합니다.

리스트의 주요 특징:

  • 대괄호 [ ]로 표현
  • 다양한 자료형을 함께 저장 가능
  • 순서가 있어 인덱싱, 슬라이싱 가능
  • 가변(mutable) 객체로 수정 가능
  • 중첩 리스트(리스트 안의 리스트) 가능
인덱스 0 42
인덱스 1 "Python"
인덱스 2 True
인덱스 3 [1, 2]
my_list = [42, "Python", True, [1, 2]]
주요 리스트 메서드
append() 리스트 끝에 요소 추가
insert() 지정한 위치에 요소 삽입
remove() 특정 값을 가진 요소 제거
pop() 지정 인덱스 요소 반환 후 제거
sort() 리스트 요소를 정렬
reverse() 리스트 요소 순서 뒤집기
리스트 생성 및 접근

리스트는 여러 가지 방법으로 생성할 수 있습니다:

# 빈 리스트 생성
empty_list = []
empty_list2 = list()

# 값이 있는 리스트 생성
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# 리스트 컴프리헨션으로 생성
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]

리스트의 요소는 인덱스를 통해 접근할 수 있습니다:

# 인덱스로 접근 (0부터 시작)
fruits = ["사과", "바나나", "오렌지"]
first_fruit = fruits[0]  # "사과"

# 음수 인덱스 (뒤에서부터 접근)
last_fruit = fruits[-1]  # "오렌지"

# 요소 값 변경
fruits[1] = "키위"  # ["사과", "키위", "오렌지"]
리스트의 활용
리스트 연결: + 연산자를 사용하여 두 리스트를 이어붙일 수 있습니다.
list1 + list2
리스트 반복: * 연산자를 사용하여 리스트를 반복할 수 있습니다.
list1 * 3
멤버십 테스트: in 연산자로 특정 요소가 리스트에 있는지 확인할 수 있습니다.
element in my_list
리스트 함수: len(), max(), min(), sum() 등의 내장 함수를 사용할 수 있습니다.
len(my_list), sum(numbers)

리스트 활용 종합 예제
# 리스트 활용 종합 예제

# 1. 리스트 생성
fruits = ["사과", "바나나", "오렌지", "포도"]
print(f"과일 리스트: {fruits}")

# 2. 인덱스로 접근
first_fruit = fruits[0]  # 첫 번째 요소
last_fruit = fruits[-1]  # 마지막 요소
print(f"첫 번째 과일: {first_fruit}, 마지막 과일: {last_fruit}")

# 3. 리스트 수정
fruits[1] = "키위"  # 두 번째 요소 변경
print(f"수정된 리스트: {fruits}")

# 4. 리스트 메서드 사용
fruits.append("망고")  # 리스트 끝에 요소 추가
print(f"요소 추가 후: {fruits}")

fruits.insert(2, "레몬")  # 지정 위치에 요소 삽입
print(f"요소 삽입 후: {fruits}")

removed_fruit = fruits.pop(3)  # 인덱스 3의 요소 제거하고 반환
print(f"제거된 과일: {removed_fruit}, 제거 후 리스트: {fruits}")

# 5. 리스트 연산
more_fruits = ["자몽", "체리"]
combined = fruits + more_fruits  # 리스트 연결
print(f"연결된 리스트: {combined}")

repeated = more_fruits * 3  # 리스트 반복
print(f"반복된 리스트: {repeated}")

# 6. 정렬과 기타 메서드
combined.sort()  # 알파벳 순서로 정렬
print(f"정렬된 리스트: {combined}")

combined.reverse()  # 리스트 역순으로 뒤집기
print(f"역순 리스트: {combined}")

# 7. 리스트 컴프리헨션
numbers = range(1, 6)
squares = [x**2 for x in numbers]
print(f"제곱 리스트: {squares}")
