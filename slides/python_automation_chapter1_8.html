단순 반복업무 자동화
파이썬의 PyAutoGUI 라이브러리를 활용하면 마우스와 키보드 동작을 프로그래밍 방식으로 제어할 수 있어, 단순하지만 반복적인 업무를 자동화할 수 있습니다. 클릭, 키 입력, 화면 캡처 등 다양한 기능을 통해 사용자의 행동을 시뮬레이션합니다.

PyAutoGUI 주요 기능

마우스 제어
마우스 이동, 클릭, 드래그, 스크롤 등의 동작을 자동화합니다.
키보드 제어
키 입력, 단축키, 특수 키 조합 등을 자동으로 입력합니다.
화면 캡처
화면 스크린샷 및 특정 이미지 인식 기능을 제공합니다.
시간 지연
작업 사이에 대기 시간을 설정하여 안정적인 자동화를 구현합니다.
이미지 인식
화면에서 특정 이미지를 찾아 위치를 파악하고 동작을 수행합니다.
창 관리
애플리케이션 창의 크기 조절, 활성화, 위치 이동 등을 수행합니다.

활용 사례

데이터 입력 자동화
여러 시스템에 동일한 데이터를 자동으로 입력하여 시간 절약
파일 다운로드 자동화
웹사이트에서 여러 파일을 순차적으로 다운로드하는 작업 자동화
애플리케이션 테스트
GUI 애플리케이션의 반복적인 테스트 시나리오 자동 실행
게임 자동화
반복적인 게임 작업을 자동화하여 효율적인 플레이 가능
주의사항

PyAutoGUI는 실행 중에 마우스와 키보드를 직접 제어하므로, 스크립트에 오류가 있을 경우 시스템 제어가 불가능해질 수 있습니다. 항상 다음 안전장치를 마련하세요:

  • FAILSAFE 기능 활성화 - 마우스를 화면 모서리로 이동하면 중지
  • 적절한 대기 시간(sleep) 설정
  • 소규모 테스트 후 전체 자동화 구현

"PyAutoGUI를 이용해 마우스를 제어하고 클릭하는 기본 예제 코드를 작성해주세요."
# PyAutoGUI를 활용한 마우스 제어 기본 예제 import pyautogui import time # 안전 장치 설정 - 화면 모서리로 마우스를 이동하면 프로그램 중지 pyautogui.FAILSAFE = True # 현재 화면 해상도 확인 screen_width, screen_height = pyautogui.size() print(f"화면 해상도: {screen_width}x{screen_height}") # 현재 마우스 위치 확인 current_x, current_y = pyautogui.position() print(f"현재 마우스 위치: x={current_x}, y={current_y}") # 마우스 이동 (절대 위치) print("화면 중앙으로 마우스 이동...") pyautogui.moveTo(screen_width // 2, screen_height // 2, duration=1) time.sleep(1) # 마우스 이동 (상대 위치) print("현재 위치에서 상대적으로 이동...") pyautogui.moveRel(100, -100, duration=1) # 오른쪽으로 100, 위로 100 픽셀 이동 time.sleep(1) # 마우스 클릭 print("클릭 수행...") pyautogui.click() # 현재 위치 클릭 time.sleep(1) # 특정 위치로 이동 후 클릭 print("특정 위치로 이동 후 클릭...") pyautogui.click(screen_width // 4, screen_height // 4, duration=1) time.sleep(1) # 더블 클릭 print("더블 클릭 수행...") pyautogui.doubleClick() time.sleep(1) # 마우스 드래그 print("드래그 수행...") pyautogui.dragTo(screen_width // 2, screen_height // 2, duration=1) time.sleep(1) print("마우스 제어 예제 완료!")

# PyAutoGUI를 활용한 키보드 제어 예제 import pyautogui import time # 안전 장치 설정 pyautogui.FAILSAFE = True # 실행 사이에 약간의 지연 추가 (너무 빠른 입력 방지) pyautogui.PAUSE = 0.5 # 메모장 실행 (Windows) print("메모장 실행 중...") pyautogui.hotkey('win', 'r') # Win+R 실행 대화상자 열기 pyautogui.typewrite('notepad') # 'notepad' 입력 pyautogui.press('enter') # Enter 키 누르기 time.sleep(1) # 메모장이 열릴 때까지 대기 # 텍스트 입력 print("텍스트 입력 중...") pyautogui.typewrite('안녕하세요! PyAutoGUI로 자동화된 텍스트입니다.\n\n') time.sleep(0.5) # 특수 키 사용 pyautogui.typewrite('특수 키 예제:\n') pyautogui.hotkey('shift', '1') # Shift+1 = ! pyautogui.typewrite(' 느낌표를 입력했습니다.\n') # 화살표 키 사용 print("화살표 키 사용...") pyautogui.press('up') # 위쪽 화살표 pyautogui.press('up') time.sleep(0.5) # 텍스트 선택 print("텍스트 선택 중...") pyautogui.hotkey('shift', 'end') # 현재 줄 끝까지 선택 time.sleep(0.5) # 선택한 텍스트 복사 print("복사 및 붙여넣기...") pyautogui.hotkey('ctrl', 'c') # 복사 pyautogui.press('end') # 줄 끝으로 이동 pyautogui.press('enter') # 새 줄 pyautogui.press('enter') # 한 줄 더 pyautogui.typewrite('복사한 텍스트: ') pyautogui.hotkey('ctrl', 'v') # 붙여넣기 print("키보드 제어 예제 완료!")

안녕하세요! PyAutoGUI로 자동화된 텍스트입니다. 특수 키 예제: ! 느낌표를 입력했습니다.
Tip: 화면에서 특정 위치의 좌표를 쉽게 찾으려면 pyautogui.displayMousePosition() 함수를 사용해보세요. 실시간으로 마우스 좌표, RGB 색상값을 보여주어 자동화 스크립트 작성 시 매우 유용합니다. 또한 이미지 인식을 이용하면 화면의 특정 요소를 찾아 클릭하는 더 안정적인 자동화가 가능합니다.
