파이썬 프로그램 실행 파일 만들기
PyInstaller를 사용한 실행 파일 생성
# PyInstaller 설치
pip install pyinstaller

# 기본 실행 파일 생성
pyinstaller your_script.py

# 단일 파일로 생성
pyinstaller --onefile your_script.py

# GUI 프로그램용 (콘솔창 없음)
pyinstaller --noconsole --onefile your_script.py

환경 준비
- PyInstaller 설치
- 필요한 모든 라이브러리가 설치되어 있는지 확인
- 프로젝트 파일 구조 정리
02
실행 파일 생성
- 적절한 PyInstaller 옵션 선택
- 리소스 파일 포함 여부 결정
- 테스트 환경에서 빌드 실행
03
배포 및 테스트
- 다른 환경에서 실행 테스트
- 필요한 추가 파일 확인
- 실행 파일 크기 최적화
주요 PyInstaller 옵션
--onefile : 단일 실행 파일 생성
--noconsole : 콘솔창 숨김 (GUI 앱용)
--icon=path : 실행 파일 아이콘 지정
--add-data : 추가 리소스 파일 포함
