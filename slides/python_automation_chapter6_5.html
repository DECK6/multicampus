Selenium을 활용한 웹 브라우저 자동화의 첫 단계

Selenium 설치 및 설정

  • 1
    파이썬 패키지 설치:
    pip install selenium
  • 2
    웹 드라이버 다운로드: 브라우저와 버전에 맞는 WebDriver 설치
  • 3
    자동 WebDriver 관리 패키지 설치: pip install webdriver-manager

브라우저 드라이버 설정

Python from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager
from selenium.webdriver.chrome.service import Service

# Chrome 드라이버 자동 설정
service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service)

# 페이지 로드
driver.get("https://www.example.com")
Selenium으로 웹 브라우저를 제어하는 기본 명령어들을 알려줘
명령어 설명 예시
driver.get(url) 지정된 URL로 이동합니다 driver.get("https://www.naver.com")
driver.title 현재 페이지의 제목을 반환합니다 title = driver.title
driver.current_url 현재 페이지의 URL을 반환합니다 current_url = driver.current_url
driver.back() 브라우저 뒤로 가기 기능을 실행합니다 driver.back()
driver.forward() 브라우저 앞으로 가기 기능을 실행합니다 driver.forward()
driver.refresh() 현재 페이지를 새로고침합니다 driver.refresh()
driver.close() 현재 탭만 닫습니다 driver.close()
driver.quit() 브라우저 창 전체를 종료합니다 driver.quit()
