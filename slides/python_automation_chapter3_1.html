파이썬 데이터 구조 개요

데이터 구조는 데이터를 효율적으로 저장, 관리, 처리하기 위한 기본 도구입니다. 업무 자동화에서 올바른 데이터 구조 선택은 코드의 효율성과 가독성을 크게 향상시킵니다.

리스트 (Lists)

순서가 있는 컬렉션으로, 다양한 데이터 타입을 포함할 수 있고 변경 가능(mutable)합니다.

numbers = [1, 2, 3, 4, 5]
mixed = [1, "Hello", 3.14, True]
딕셔너리 (Dictionaries)

키-값 쌍으로 이루어진 컬렉션으로, 빠른 데이터 검색과 매핑에 유용합니다.

person = {"name": "홍길동", "age": 30}
print(person["name"]) # 홍길동
튜플 (Tuples)

순서가 있는 컬렉션이지만, 리스트와 달리 변경 불가능(immutable)합니다.

coordinates = (10, 20)
rgb = (255, 100, 50)
세트 (Sets)

중복이 없는 고유 항목의 컬렉션으로, 집합 연산이 가능합니다.

fruits = {"apple", "banana", "cherry"}
print("apple" in fruits) # True
배열 (Arrays)

NumPy 라이브러리를 통해 제공되는 배열로, 수치 계산에 최적화되어 있습니다.

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
데이터프레임 (DataFrames)

판다스(Pandas) 라이브러리의 2차원 테이블 구조로, 행과 열이 있으며 데이터 분석과 조작에 가장 많이 사용됩니다. 엑셀 스프레드시트와 유사하지만 더 강력한 기능을 가지고 있으며, 업무 자동화에서 가장 중요한 데이터 구조 중 하나입니다.

import pandas as pd
df = pd.DataFrame({
  "이름": ["홍길동", "김철수", "이영희"],
  "나이": [30, 25, 28],
  "직업": ["개발자", "디자이너", "마케터"]
})
"데이터 구조를 올바르게 선택하는 것은 효율적인 코드의 첫 걸음입니다"
