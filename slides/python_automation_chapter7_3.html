행동 1: 브라우저 열기

웹 자동화의 첫 번째 단계는 브라우저를 열고 제어할 수 있는 환경을 설정하는 것입니다. 파이썬에서 Selenium을 사용하여 다양한 브라우저를 열고 제어할 수 있습니다.

설치 및 환경 설정

1
Selenium 라이브러리 설치하기
# ChatGPT: "파이썬에서 셀레니움 설치 명령어"
pip install selenium
2
WebDriver Manager 설치하기
# ChatGPT: "웹드라이버 매니저 설치 명령어"
pip install webdriver-manager
3
필요한 모듈 임포트하기
from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager
from selenium.webdriver.chrome.service import Service
왜 WebDriver Manager를 사용해야 할까요?
WebDriver Manager는 브라우저 드라이버 파일을 자동으로 다운로드하고 관리해줍니다. 이를 사용하면 브라우저 버전이 업데이트되어도 항상 호환되는 드라이버를 자동으로 설치해주므로 코드 유지보수가 쉬워집니다.

브라우저 열기

기본 브라우저 열기

# ChatGPT: "셀레니움으로 크롬 브라우저 열기"
from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager
from selenium.webdriver.chrome.service import Service

# 서비스 객체 생성 및 드라이버 자동 설치
service = Service(ChromeDriverManager().install())

# 브라우저 드라이버 객체 생성
driver = webdriver.Chrome(service=service)

# 웹사이트로 이동
driver.get("https://www.example.com")

헤드리스 모드로 열기

# ChatGPT: "셀레니움 헤드리스 모드 설정"
from selenium.webdriver.chrome.options import Options

options = Options()
options.add_argument("--headless=new")

driver = webdriver.Chrome(
  service=Service(ChromeDriverManager().install()),
  options=options
)

다양한 브라우저 옵션 설정하기

# ChatGPT: "셀레니움 크롬 브라우저 세부 옵션 설정"
options = Options()

# 창 최대화
options.add_argument("--start-maximized")

# 팝업 차단 비활성화
options.add_argument("--disable-popup-blocking")
# 브라우저 알림 비활성화
options.add_argument("--disable-notifications")

# 확장 프로그램 로드하기
options.add_extension('path/to/extension.crx')

# 특정 크기로 창 열기
options.add_argument("--window-size=1920,1080")
