행동 6: 페이지 스크롤

웹페이지는 종종 화면보다 길어 스크롤이 필요합니다. Selenium을 사용하면 자바스크립트를 통해 페이지를 다양한 방식으로 스크롤할 수 있으며, 이는 화면 밖 콘텐츠에 접근하거나 무한 스크롤 페이지를 자동화하는 데 필수적입니다.

기본 스크롤 방법

자바스크립트를 통해 페이지의 다양한 위치로 스크롤할 수 있습니다.

# ChatGPT: "셀레니움으로 페이지 스크롤하기"
# 페이지 맨 아래로 스크롤
driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")

# 페이지 맨 위로 스크롤
driver.execute_script("window.scrollTo(0, 0);")

# 특정 픽셀만큼 아래로 스크롤
driver.execute_script("window.scrollBy(0, 500);")

# 특정 픽셀만큼 위로 스크롤
driver.execute_script("window.scrollBy(0, -300);")

특정 요소로 스크롤

페이지 내의 특정 요소가 보이도록 스크롤하는 방법입니다.

# ChatGPT: "셀레니움으로 특정 요소가 보이도록 스크롤하기"
from selenium import webdriver
from selenium.webdriver.common.by import By

# 스크롤할 요소 찾기
element = driver.find_element(By.ID, "footer")

# 요소가 화면에 보이도록 스크롤
driver.execute_script("arguments[0].scrollIntoView();", element)

# 요소가 화면 상단에 위치하도록 스크롤
driver.execute_script("arguments[0].scrollIntoView(true);", element)

# 요소가 화면 하단에 위치하도록 스크롤
driver.execute_script("arguments[0].scrollIntoView(false);", element)

무한 스크롤 페이지 처리

소셜 미디어, 검색 결과 등 많은 웹사이트에서 무한 스크롤을 사용합니다. 이를 자동화하는 방법입니다.

# ChatGPT: "셀레니움으로 무한 스크롤 페이지 자동화하기"
import time

def scroll_to_bottom(driver, scroll_pause_time=1.0):
  # 현재 스크롤 위치 가져오기
  last_height = driver.execute_script("return document.body.scrollHeight")
  
  while True:
    # 페이지 맨 아래로 스크롤
    driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")
    
    # 페이지 로딩 대기
    time.sleep(scroll_pause_time)
    
    # 새 스크롤 높이 계산
    new_height = driver.execute_script("return document.body.scrollHeight")
    
    # 스크롤이 더 이상 내려가지 않으면 종료
    if new_height == last_height:
      break
    last_height = new_height

특정 데이터 수집을 위한 스크롤

# ChatGPT: "셀레니움으로 스크롤하며 데이터 수집하기"
def scroll_and_collect_data(driver, target_count=50):
  all_items = []
  
  while len(all_items) < target_count:
    # 현재 표시된 아이템 수집
    items = driver.find_elements(By.CSS_SELECTOR, ".product-item")
    
    # 새 아이템만 목록에 추가
    for item in items:
      item_id = item.get_attribute("data-id")
      if item_id not in [i["id"] for i in all_items]:
        item_data = {
          "id": item_id,
          "name": item.find_element(By.CSS_SELECTOR, ".item-name").text,
          "price": item.find_element(By.CSS_SELECTOR, ".item-price").text
        }
        all_items.append(item_data)
    
    print(f"현재 {len(all_items)}개 아이템 수집됨")
    
    # 더 많은 아이템이 필요하면 스크롤
    if len(all_items) < target_count:
      driver.execute_script("window.scrollBy(0, 800);")
      time.sleep(1) # 새 콘텐츠 로딩 대기
  
  return all_items
스크롤 팁

1. 너무 빠른 스크롤은 콘텐츠가 제대로 로드되지 않을 수 있으니 적절한 대기 시간을 설정하세요.

2. 스크롤 전후로 페이지 높이를 비교하면 새 콘텐츠가 로드되었는지 확인할 수 있습니다.

3. 특정 요소가 보이도록 스크롤한 후 클릭이나 데이터 추출 작업을 수행하세요.
