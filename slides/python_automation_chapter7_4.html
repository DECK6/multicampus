행동 2: 웹사이트 접속

브라우저를 실행한 후, 다음 단계는 웹사이트에 접속하는 것입니다. Selenium은 다양한 방법으로 웹페이지에 접속하고 페이지 로딩 상태를 확인할 수 있는 기능을 제공합니다.

기본 웹사이트 접속 방법

웹사이트에 접속하는 가장 기본적인 방법은 get() 메서드를 사용하는 것입니다.

# ChatGPT: "셀레니움으로 웹사이트 접속하기"
from selenium import webdriver
from webdriver_manager.chrome import ChromeDriverManager
from selenium.webdriver.chrome.service import Service

# 브라우저 드라이버 설정
service = Service(ChromeDriverManager().install())
driver = webdriver.Chrome(service=service)

# 웹사이트 접속
driver.get("https://www.example.com")

# 현재 URL 확인
current_url = driver.current_url
print(f"현재 페이지: {current_url}")
get() 메서드는 페이지가 완전히 로드될 때까지 대기합니다. 그러나 JavaScript로 로드되는 동적 콘텐츠는 기다리지 않을 수 있습니다.

페이지 로딩 기다리기

웹페이지의 요소가 로드될 때까지 명시적으로 기다리는 방법:

# ChatGPT: "셀레니움 페이지 로딩 대기 코드"
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# 웹사이트 접속
driver.get("https://www.example.com")

# 특정 요소가 나타날 때까지 최대 10초 기다림
try:
  element = WebDriverWait(driver, 10).until(
    EC.presence_of_element_located((By.ID, "example-element-id"))
  )
  print("페이지가 성공적으로 로드되었습니다!")
except:
  print("페이지 로딩 시간이 초과되었습니다.")
암시적 대기와 명시적 대기의 차이

암시적 대기(Implicit Wait): 모든 요소에 대해 전역적으로 적용되는 대기 시간을 설정합니다.

driver.implicitly_wait(10) # 초 단위

명시적 대기(Explicit Wait): 특정 조건이 만족될 때까지 대기하며, 특정 요소에만 적용됩니다.

페이지 로딩 확인하기

# ChatGPT: "셀레니움으로 페이지 로딩 상태 확인하기"
def is_page_loaded(driver):
  # 자바스크립트 실행으로 페이지 로딩 상태 확인
  return driver.execute_script("return document.readyState") == "complete"

# 사용 예시
driver.get("https://www.example.com")

# 페이지 로딩 확인
if is_page_loaded(driver):
  print("페이지가 완전히 로드되었습니다.")
else:
  print("페이지 로딩 중...")
