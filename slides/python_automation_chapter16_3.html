워드클라우드 생성하기

크롤링한 뉴스 제목들을 활용하여 시각적으로 표현하는 워드클라우드를 생성합니다.

한글 워드클라우드 생성 코드

from wordcloud import WordCloud
from konlpy.tag import Okt
from collections import Counter

def create_wordcloud(texts):
    # 형태소 분석기 초기화
    okt = Okt()
    
    # 명사 추출
    nouns = []
    for text in texts:
        nouns.extend(okt.nouns(text))
    
    # 빈도수 계산
    word_count = Counter(nouns)
    
    # 워드클라우드 생성
    wc = WordCloud(
        font_path='malgun.ttf',     # 한글 폰트 경로
        background_color='white',
        width=800,
        height=600,
        max_words=100
    )
    
    # 워드클라우드 생성 및 저장
    wc.generate_from_frequencies(word_count)
    wc.to_file('wordcloud.png')

주요 설정 옵션:

  • font_path: 한글 표시를 위한 폰트 파일 경로
  • background_color: 배경색 설정
  • width/height: 이미지 크기 설정
  • max_words: 최대 표시 단어 수
