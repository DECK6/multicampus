HTML 선택자의 이해
웹 자동화에서 요소를 찾기 위한 선택자(Selector)의 종류와 활용법

CSS 선택자

웹페이지에서 스타일을 적용할 요소를 선택하는 패턴으로, 자동화 도구에서도 요소를 찾는 데 널리 사용됩니다.

선택자 설명 예시
#id ID로 요소 선택 #login-button
.class 클래스로 요소 선택 .navbar-item
tag 태그로 요소 선택 div
[attr=value] 속성으로 요소 선택 [type="submit"]

XPath 선택자

XML 문서의 노드를 선택하는 경로 표현식으로, HTML 구조를 탐색하는 강력한 방법을 제공합니다.

선택자 설명 예시
//tag 모든 태그 선택 //div
//tag[@attr='value'] 속성값으로 선택 //button[@id='submit']
//tag[contains(text(),'내용')] 텍스트로 선택 //a[contains(text(),'로그인')]
//parent/child 부모-자식 관계로 선택 //ul/li
파이썬으로 셀레니움에서 CSS 선택자와 XPath를 사용하는 방법을 알려줘
Python from selenium import webdriver
from selenium.webdriver.common.by import By

# 브라우저 드라이버 초기화
driver = webdriver.Chrome()
driver.get("https://example.com")

# CSS 선택자로 요소 찾기
login_button = driver.find_element(By.CSS_SELECTOR, "#login-button")
items = driver.find_elements(By.CSS_SELECTOR, ".item")

# XPath로 요소 찾기
search_box = driver.find_element(By.XPATH, "//input[@name='search']")
nav_links = driver.find_elements(By.XPATH, "//nav//a")

# 텍스트로 요소 찾기 (XPath 이용)
signup_link = driver.find_element(By.XPATH, "//a[contains(text(), '회원가입')]")
"선택자를 활용한 웹 요소 탐색은 효과적인 웹 자동화의 핵심입니다."
