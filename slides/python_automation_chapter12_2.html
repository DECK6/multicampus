gspread 기본 사용법
스프레드시트 접근 및 데이터 읽기
ChatGPT: "Show me how to read data from Google Spreadsheet using gspread"
import gspread
from oauth2client.service_account import ServiceAccountCredentials

# 스프레드시트 열기
gc = gspread.authorize(credentials)
worksheet = gc.open('스프레드시트_이름').sheet1

# 데이터 읽기
all_values = worksheet.get_all_values() # 전체 데이터
cell_value = worksheet.acell('A1').value # 특정 셀
row_values = worksheet.row_values(1) # 특정 행
gspread는 다양한 방식으로 스프레드시트 데이터에 접근할 수 있습니다:
  • get_all_values(): 전체 시트의 데이터를 2차원 리스트로 반환
  • acell(): 특정 셀의 값을 반환 (예: 'A1', 'B2')
  • row_values(): 특정 행의 모든 값을 리스트로 반환
스프레드시트 이름은 구글 드라이브에서 확인할 수 있는 문서의 실제 이름을 사용해야 합니다. 공유된 스프레드시트의 경우 이메일 주소로도 접근이 가능합니다.
