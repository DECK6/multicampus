유용한 파이썬 라이브러리
데이터 처리 라이브러리
Pandas
pip install pandas
데이터 분석 및 조작을 위한 강력한 라이브러리로, 엑셀 파일 읽기/쓰기, 데이터 필터링 등의 작업에 유용합니다.
import pandas as pd
df = pd.read_excel('data.xlsx')
filtered = df[df['금액'] > 10000]
NumPy
pip install numpy
대규모 다차원 배열과 행렬 연산을 지원하며, 수치 계산 작업에 필수적인 라이브러리입니다.
import numpy as np
data = np.array([1, 2, 3, 4, 5])
result = np.mean(data) # 평균 계산
openpyxl
pip install openpyxl
엑셀 파일(.xlsx)을 읽고 쓸 수 있는 라이브러리로, 서식과 수식을 포함한 고급 엑셀 작업이 가능합니다.
from openpyxl import Workbook
wb = Workbook()
wb.save("output.xlsx")
python-docx
pip install python-docx
워드 문서(.docx)를 생성하고 수정할 수 있는 라이브러리로, 보고서 자동 생성에 유용합니다.
from docx import Document
doc = Document()
doc.add_heading('보고서 제목', 0)
자동화 라이브러리
Selenium
pip install selenium
웹 브라우저를 자동화하는 라이브러리로, 웹 크롤링, 폼 자동 제출 등에 사용됩니다.
from selenium import webdriver
driver = webdriver.Chrome()
driver.get("https://example.com")
PyAutoGUI
pip install pyautogui
마우스와 키보드를 제어하여 GUI 기반 응용 프로그램을 자동화하는 라이브러리입니다.
import pyautogui
pyautogui.click(100, 100)
pyautogui.write('Hello World')
Requests
pip install requests
HTTP 요청을 간편하게 만들 수 있는 라이브러리로, API 호출 및 웹 서비스 연동에 유용합니다.
import requests
response = requests.get('https://api.example.com/data')
data = response.json()
Beautiful Soup
pip install beautifulsoup4
HTML 및 XML 파일에서 데이터를 추출하는 라이브러리로, 웹 스크래핑에 널리 사용됩니다.
from bs4 import BeautifulSoup
soup = BeautifulSoup(html_doc, 'html.parser')
titles = soup.find_all('h2')
시각화 및 보고서 라이브러리
Matplotlib
pip install matplotlib
다양한 차트와 그래프를 생성할 수 있는 기본 시각화 라이브러리입니다.
import matplotlib.pyplot as plt
plt.plot([1, 2, 3, 4], [10, 20, 25, 30])
plt.savefig('chart.png')
Seaborn
pip install seaborn
Matplotlib 기반의 고급 시각화 라이브러리로, 통계 차트를 쉽게 생성할 수 있습니다.
import seaborn as sns
sns.set_theme()
sns.barplot(x="부서", y="매출", data=df)
ReportLab
pip install reportlab
PDF 파일을 생성하고 편집할 수 있는 라이브러리로, 복잡한 보고서 자동화에 적합합니다.
from reportlab.pdfgen import canvas
c = canvas.Canvas("report.pdf")
c.drawString(100, 750, "자동 생성 보고서")
Plotly
pip install plotly
대화형 그래프와 대시보드를 만들 수 있는 라이브러리로, 동적 보고서에 유용합니다.
import plotly.express as px
fig = px.bar(df, x="월", y="판매량")
fig.write_html("sales_report.html")
기타 유용한 라이브러리
schedule
pip install schedule
작업을 예약하고 주기적으로 실행할 수 있는 라이브러리로, 간단한 작업 스케줄링에 적합합니다.
import schedule
schedule.every().day.at("10:00").do(morning_report)
schedule.every().monday.do(weekly_report)
smtplib
파이썬 내장 라이브러리
SMTP 프로토콜을 사용하여 이메일을 보낼 수 있는 내장 라이브러리입니다.
import smtplib
with smtplib.SMTP('smtp.gmail.com', 587) as server:
  server.send_message(msg)
pathlib
파이썬 내장 라이브러리
파일 경로를 객체로 다루는 현대적인 방식의 파일 시스템 경로 라이브러리입니다.
from pathlib import Path
data_dir = Path("data")
files = list(data_dir.glob("*.xlsx"))
logging
파이썬 내장 라이브러리
자동화 스크립트에서 로그를 기록하기 위한 내장 라이브러리입니다.
import logging
logging.basicConfig(filename='app.log', level=logging.INFO)
logging.info('작업이 완료되었습니다')
라이브러리 활용 팁
라이브러리의 공식 문서를 참조하면 더 많은 기능과 활용 예제를 확인할 수 있습니다.
여러 라이브러리를 조합하여 사용하면 복잡한 업무 프로세스도 자동화할 수 있습니다.
중요한 자동화 스크립트는 가상 환경을 통해 라이브러리 버전을 독립적으로 관리하세요.
"적절한 라이브러리를 활용하면 몇 시간 걸릴 업무를 몇 분 안에 처리할 수 있습니다."
