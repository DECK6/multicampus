행동 4: 클릭하기

웹 요소를 찾은 후, 가장 기본적인 상호작용은 클릭하는 것입니다. 셀레니움은 다양한 클릭 방식을 지원하여 실제 사용자의 행동을 시뮬레이션할 수 있습니다.

기본 클릭 방법

요소를 찾은 후 가장 간단하게 클릭하는 방법은 click() 메서드를 사용하는 것입니다.

# ChatGPT: "셀레니움으로 버튼 클릭하기"
from selenium import webdriver
from selenium.webdriver.common.by import By

# 브라우저 드라이버 설정 (생략)

# 요소 찾기
login_button = driver.find_element(By.ID, "login-btn")

# 요소 클릭하기
login_button.click()

# 한 줄로 요소 찾고 바로 클릭하기
driver.find_element(By.CSS_SELECTOR, ".submit-button").click()

다양한 클릭 액션

ActionChains를 사용하여 더 복잡한 클릭 동작을 구현할 수 있습니다:

# ChatGPT: "셀레니움 ActionChains로 다양한 클릭 구현하기"
from selenium.webdriver.common.action_chains import ActionChains

# 요소 찾기
element = driver.find_element(By.ID, "context-menu-element")

# ActionChains 객체 생성
actions = ActionChains(driver)

# 우클릭(컨텍스트 클릭) 수행
actions.context_click(element).perform()

# 더블 클릭 수행
double_click_element = driver.find_element(By.ID, "double-click-element")
actions.double_click(double_click_element).perform()

클릭 문제 해결하기

일반적인 클릭 문제
  • 요소가 화면에 보이지 않아 클릭할 수 없음
  • 다른 요소가 클릭하려는 요소를 가리고 있음
  • 요소가 비활성화되어 클릭할 수 없음
  • 요소가 아직 로드되지 않아 클릭할 수 없음
# ChatGPT: "셀레니움에서 요소를 보이게 스크롤한 후 클릭하기"
# 요소가 보이지 않는 경우 스크롤하여 보이게 하기
element = driver.find_element(By.ID, "element-at-bottom")

# 자바스크립트로 요소가 보이도록 스크롤
driver.execute_script("arguments[0].scrollIntoView();", element)

# 스크롤 후 잠시 대기
import time
time.sleep(0.5)

# 이제 클릭 수행
element.click()

눌렀던 마우스 버튼 놓기
# ChatGPT: "셀레니움에서 자바스크립트로 클릭하기"
# 기본 클릭이 작동하지 않을 때 자바스크립트로 클릭
element = driver.find_element(By.ID, "difficult-button")
driver.execute_script("arguments[0].click();", element)
클릭 전 대기하기

클릭하기 전에 항상 요소가 클릭 가능한 상태인지 확인하는 것이 좋습니다. 명시적 대기(Explicit Wait)를 사용하여 요소가 클릭 가능할 때까지 기다립니다.

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# 요소가 클릭 가능할 때까지 최대 10초 대기
element = WebDriverWait(driver, 10).until(
  EC.element_to_be_clickable((By.ID, "clickable-button"))
)
# 이제 안전하게 클릭
element.click()
