변수와 자료형 개요
변수란?

변수는 데이터를 저장하는 메모리 공간을 가리키는 이름입니다. 파이썬에서는 변수를 선언할 때 별도의 타입을 명시하지 않아도 됩니다.

변수 선언 방법은 매우 간단합니다:

변수명 = 값

파이썬은 동적 타이핑(Dynamic Typing) 언어로, 변수에 할당된 값에 따라 자료형이 자동으로 결정됩니다.

변수 이름 규칙
  • 문자, 숫자, 밑줄(_)을 사용할 수 있습니다.
  • 숫자로 시작할 수 없습니다.
  • 대소문자를 구분합니다 (age ≠ Age).
  • 예약어(if, for, while 등)는 사용할 수 없습니다.
  • 보통 스네이크 케이스(snake_case)를 사용합니다.
자료형이란?

자료형(Data Type)은 데이터가 메모리에 저장되는 방식과 이 데이터로 수행할 수 있는 연산을 결정합니다.

파이썬의 자료형은 크게 기본 자료형과 컬렉션 자료형으로 나눌 수 있습니다.

특정 값의 자료형을 확인하려면 type() 함수를 사용할 수 있습니다.

int (정수)
소수점이 없는 숫자입니다.
age = 25
float (실수)
소수점이 있는 숫자입니다.
pi = 3.14159
str (문자열)
텍스트 데이터입니다.
name = "파이썬"
bool (논리)
True 또는 False 값입니다.
is_valid = True
list (리스트)
순서가 있는 컬렉션입니다.
numbers = [1, 2, 3]
dict (딕셔너리)
키-값 쌍의 컬렉션입니다.
user = {"name": "Kim"}

변수와 자료형 활용 예제
# 다양한 자료형의 변수 선언과 사용 # 기본 자료형 name = "홍길동" # 문자열(str) 타입 age = 30 # 정수(int) 타입 height = 175.5 # 실수(float) 타입 is_student = False # 논리(bool) 타입 nothing = None # None 타입 # 자료형 확인하기 print(f"name의 자료형: {type(name)}") # 출력: name의 자료형: print(f"age의 자료형: {type(age)}") # 출력: age의 자료형: print(f"height의 자료형: {type(height)}") # 출력: height의 자료형: # 동적 타이핑 - 동일한 변수에 다른 타입 할당 가능 variable = 10 # 정수 타입 print(f"variable: {variable}, 타입: {type(variable)}") variable = "파이썬" # 문자열 타입으로 변경 print(f"variable: {variable}, 타입: {type(variable)}") # 컬렉션 자료형 numbers = [1, 2, 3, 4, 5] # 리스트(list) point = (10, 20) # 튜플(tuple) user_info = {"name": "김철수", "age": 25} # 딕셔너리(dict) unique_numbers = {1, 2, 3, 3, 4} # 집합(set) - 중복 제거됨 print(f"numbers: {numbers}, 타입: {type(numbers)}") print(f"unique_numbers: {unique_numbers}") # 출력: unique_numbers: {1, 2, 3, 4}
