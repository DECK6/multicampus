인터넷 1분만에 이해하기

클라이언트

웹 브라우저를 통해 서버에 정보를 요청하고 받아온 정보를 화면에 표시합니다.

인터넷

클라이언트와 서버 간의 데이터를 주고받는 전 세계적 네트워크입니다.

서버

요청을 받아 처리하고 HTML, CSS, JavaScript 등의 파일을 클라이언트에게 전송합니다.

HTTP 프로토콜

웹에서 정보를 주고받는 규약으로, 클라이언트가 GET, POST 등의 요청을 보내면 서버는 응답 코드(200, 404 등)와 함께 데이터를 반환합니다.

웹 페이지의 구성

• HTML: 웹페이지의 구조와 내용을 정의
• CSS: 디자인과 레이아웃을 담당
• JavaScript: 동적인 기능과 상호작용 처리

URL 구조

프로토콜(http://) + 도메인(www.example.com) + 경로(/page) + 매개변수(?id=123)로 구성되며, 웹에서 특정 자원의 위치를 가리킵니다.

쿠키와 세션

웹사이트와 사용자 간의 상태를 유지하는 방법으로, 로그인 상태 유지, 장바구니 정보 저장 등에 활용됩니다.

"인터넷은 클라이언트-서버 모델에 기반한 전 세계적 정보 교환 네트워크입니다."
