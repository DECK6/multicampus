주피터 노트북 설치
Jupyter Notebook은 여러 가지 방법으로 설치할 수 있습니다. 가장 일반적인 방법은 pip를 통한 직접 설치입니다. 이 방법을 통해 파이썬 자동화 개발을 위한 환경을 구축해 보겠습니다.
pip 설치
Python의 기본 패키지 관리자인 pip를 통해 필요한 패키지만 설치하는 방법으로, 더 가볍고 유연하게 환경을 구성할 수 있습니다.
  • 최소 필요 패키지만 설치 가능
  • 기존 Python 환경이 있는 경우 적합
  • 디스크 공간 절약

pip를 통한 직접 설치

1
Python 설치 확인
pip를 사용하기 위해서는 먼저 Python이 설치되어 있어야 합니다. 명령 프롬프트에서 다음 명령어로 설치를 확인합니다.
C:\Users\username> python --version
Python 3.10.5
2
pip를 사용하여 Jupyter 설치
명령 프롬프트 또는 터미널에서 다음 명령어를 실행하여 Jupyter Notebook을 설치합니다.
C:\Users\username> pip install notebook
Successfully installed notebook-6.4.12 ...
설치 확인
Jupyter Notebook이 제대로 설치되었는지 확인하려면 명령 프롬프트에서 다음 명령어를 실행해 보세요.
C:\Users\username> jupyter --version
jupyter core : 4.8.1
jupyter-notebook : 6.4.5

추가 설정 및 실행

3
JupyterLab 설치 (선택사항)
Jupyter Notebook의 더 발전된 형태인 JupyterLab을 설치하려면 다음 명령어를 실행합니다.
C:\Users\username> pip install jupyterlab
Successfully installed jupyterlab-3.4.3 ...
4
Jupyter Notebook 실행
설치가 완료되면 다음 명령어로 Jupyter Notebook을 실행합니다.
C:\Users\username> jupyter notebook

자주 발생하는 문제 및 해결방법

'jupyter' 명령어를 찾을 수 없음
Python이 시스템 경로에 제대로 추가되지 않았을 가능성이 높습니다. Python 설치 시 "Add Python to PATH" 옵션을 선택했는지 확인하거나, 다음 명령어로 직접 실행해 보세요. python -m jupyter notebook
모듈 의존성 충돌
다른 패키지와의 충돌이 발생할 수 있습니다. 가상환경을 사용하여 설치하는 것이 좋습니다. python -m venv jupyter-env jupyter-env\Scripts\activate # Windows source jupyter-env/bin/activate # macOS/Linux pip install notebook
브라우저가 자동으로 열리지 않음
Jupyter가 실행되면 터미널에 표시된 URL(일반적으로 http://localhost:8888)을 수동으로 브라우저에 입력하여 접속할 수 있습니다.
