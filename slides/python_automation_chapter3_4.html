데이터프레임 변환 및 처리

데이터프레임의 변환, 처리, 결합 작업을 통해 자동화 작업의 효율성을 높이는 방법을 알아봅니다.

데이터 변환 함수

데이터프레임의 값을 변환하는 함수들을 사용하여 데이터 처리를 자동화할 수 있습니다.

판다스에서 apply, map, applymap 함수 사용 예시를 보여줘
import pandas as pd
df = pd.DataFrame({
'금액': [1000, 2000, 3000],
'할인율': [0.1, 0.2, 0.3],
'상품명': ['A상품', 'B상품', 'C상품']
})
# apply() - 열 또는 행에 함수 적용
df['할인금액'] = df.apply(lambda row: row['금액'] * row['할인율'], axis=1)
# map() - Series에 함수 적용
가격대 = {1000: '저가', 2000: '중가', 3000: '고가'}
df['가격대'] = df['금액'].map(가격대)
변환 결과:
  금액 할인율 상품명 할인금액 가격대
0 1000 0.1 A상품 100.0 저가
1 2000 0.2 B상품 400.0 중가
2 3000 0.3 C상품 900.0 고가
그룹화와 집계

데이터를 그룹화하고 집계하여 요약 정보를 도출하는 작업을 자동화할 수 있습니다.

판다스에서 groupby를 사용한 데이터 그룹화 예시 코드
판매_데이터 = pd.DataFrame({
'부서': ['A', 'B', 'A', 'A', 'B'],
'판매액': [1000, 1500, 2000, 1200, 2500]
})
# 부서별 판매액 합계
부서별_합계 = 판매_데이터.groupby('부서')['판매액'].sum()
# 여러 집계 함수 동시 적용
부서별_통계 = 판매_데이터.groupby('부서')['판매액'].agg([
'sum', 'mean', 'count', 'max', 'min'
])
부서별 통계 결과:
    sum   mean count max min
부서
A   4200 1400.0   3 2000 1000
B   4000 2000.0   2 2500 1500
데이터프레임 병합 및 결합

여러 데이터 소스에서 얻은 데이터프레임을 병합하여 통합된 데이터셋을 생성할 수 있습니다.

판다스에서 merge, join, concat 함수를 사용한 데이터 결합 예시
# 고객 데이터
고객_df = pd.DataFrame({
'고객ID': [1, 2, 3],
'이름': ['홍길동', '김철수', '이영희']
})
# 주문 데이터
주문_df = pd.DataFrame({
'주문ID': [101, 102, 103],
'고객ID': [1, 3, 1],
'금액': [1000, 2000, 1500]
})
# merge로 두 데이터프레임 결합 (SQL JOIN과 유사)
고객주문 = pd.merge(
고객_df, 주문_df, on='고객ID', how='left'
)
# concat으로 데이터프레임 연결
df1 = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
df2 = pd.DataFrame({'A': [5, 6], 'B': [7, 8]})
수직결합 = pd.concat([df1, df2]) # 행 방향 결합
병합 유형
left join
right join
inner join
outer join

SQL 조인과 유사한 개념으로, 두 데이터셋을 연결하는 방식을 지정합니다.

데이터 정제 및 결측치 처리

실무 데이터는 결측치(NaN)나 중복 등 다양한 문제가 있으며 이를 자동으로 처리할 수 있습니다.

판다스에서 결측치와 중복 처리하는 코드 예시
데이터 = pd.DataFrame({
'A': [1, 2, None, 4],
'B': [5, None, None, 8],
'C': ['x', 'y', 'z', 'x']
})
# 결측치 확인
결측치_확인 = 데이터.isna().sum()
# 결측치 채우기
채운_데이터 = 데이터.fillna({'A': 0, 'B': 데이터['B'].mean()})
# 결측치가 있는 행 제거
완전한_데이터 = 데이터.dropna()
# 중복 행 제거
중복제거 = 데이터.drop_duplicates(subset=['C'])
업무 자동화 응용

이러한 데이터 처리 기술은 매일 반복되는 보고서 작성, 데이터 정제, 여러 파일 통합 등의 업무를 자동화하는 데 필수적입니다. 엑셀로 수작업 처리하던 작업을 몇 줄의 코드로 처리할 수 있습니다.

"판다스의 강력한 변환 기능은 복잡한 데이터 처리 작업도 몇 줄의 코드로 자동화할 수 있게 해줍니다"
