Pandas 라이브러리 소개

데이터프레임 (DataFrame)

2차원 테이블 형태의 데이터 구조로, 엑셀 시트와 유사한 형태입니다. 행과 열로 구성되며, 다양한 타입의 데이터를 저장할 수 있습니다.

import pandas as pd
df = pd.DataFrame({'A': [1, 2, 3], 'B': ['a', 'b', 'c']})

시리즈 (Series)

1차원 배열 형태의 데이터 구조로, 데이터프레임의 한 열에 해당합니다. 인덱스를 가지고 있어 데이터에 쉽게 접근할 수 있습니다.

series = pd.Series([1, 2, 3], index=['x', 'y', 'z'])

파일 입출력

Excel, CSV, JSON 등 다양한 형식의 파일을 쉽게 읽고 쓸 수 있습니다. 대용량 데이터도 효율적으로 처리할 수 있습니다.

df = pd.read_excel('data.xlsx')
df.to_csv('output.csv')

데이터 분석 기능

그룹화, 정렬, 필터링, 집계 등 다양한 데이터 분석 기능을 제공합니다. NumPy와의 완벽한 호환으로 수치 계산도 가능합니다.

grouped = df.groupby('column').mean()
filtered = df[df['value'] > 100]
