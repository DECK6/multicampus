파이썬 기본연산 - 산술연산자
파이썬에서 산술연산자는 수학적 계산을 수행하는 기호들입니다. 이 연산자들을 사용해 덧셈, 뺄셈, 곱셈, 나눗셈 등의 기본 연산을 수행할 수 있습니다. 파이썬의 산술연산자들은 직관적이고 사용하기 쉽게 설계되어 있습니다.
+
덧셈 (Addition)
두 숫자의 합을 계산합니다
a = 5 + 3 # 결과: 8
-
뺄셈 (Subtraction)
두 숫자의 차를 계산합니다
a = 5 - 3 # 결과: 2
*
곱셈 (Multiplication)
두 숫자의 곱을 계산합니다
a = 5 * 3 # 결과: 15
/
나눗셈 (Division)
나눗셈의 결과를 실수로 반환합니다
a = 5 / 2 # 결과: 2.5
//
정수 나눗셈 (Floor Division)
나눗셈의 몫을 정수로 반환합니다
a = 5 // 2 # 결과: 2
%
나머지 (Modulo)
나눗셈의 나머지를 반환합니다
a = 5 % 2 # 결과: 1
**
거듭제곱 (Exponentiation)
왼쪽 숫자를 오른쪽 숫자만큼 제곱합니다
a = 2 ** 3 # 결과: 8

산술 연산자 활용 예제
# 파이썬 산술 연산자 예제 # 변수 초기화 a = 10 b = 3 # 모든 산술 연산자 사용 덧셈 = a + b 뺄셈 = a - b 곱셈 = a * b 나눗셈 = a / b 정수나눗셈 = a // b 나머지 = a % b 거듭제곱 = a ** b # 결과 출력 print(f"덧셈 (a + b): {덧셈}") # 출력: 13 print(f"뺄셈 (a - b): {뺄셈}") # 출력: 7 print(f"곱셈 (a * b): {곱셈}") # 출력: 30 print(f"나눗셈 (a / b): {나눗셈}") # 출력: 3.3333333333333335 print(f"정수나눗셈 (a // b): {정수나눗셈}") # 출력: 3 print(f"나머지 (a % b): {나머지}") # 출력: 1 print(f"거듭제곱 (a ** b): {거듭제곱}") # 출력: 1000 # 복합 연산 예제 complex_calc = (a + b) * (a - b) / (a % b) print(f"복합 연산 결과: {complex_calc}") # 출력: 130.0
