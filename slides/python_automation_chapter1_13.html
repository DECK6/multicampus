Jupyter Notebook은 코드 작성, 실행, 결과 확인, 문서화를 한 곳에서 할 수 있는 강력한 도구입니다. 쉽게 사용할 수 있지만, 다양한 기능과 단축키를 활용하면 더욱 효율적으로 작업할 수 있습니다. 이번 슬라이드에서는 주피터 노트북의 기본 사용법과 유용한 팁들을 알아보겠습니다.

인터페이스 구성

메뉴바: 파일 관리, 셀 편집, 커널 제어 등 다양한 메뉴 제공
툴바: 자주 사용하는 기능들에 대한 바로가기 버튼
셀(Cell): 코드나 텍스트를 입력하고 실행하는 기본 단위
커널(Kernel): 코드를 실행하는 엔진, 재시작 또는 중단 가능

셀 유형

코드 셀
Python 코드를 작성하고 실행하는 셀입니다. 실행 결과는 셀 아래에 표시됩니다.
# 간단한 계산 x = 10 y = 20 print(f"합계: {x + y}")
마크다운 셀
서식이 있는 텍스트, 이미지, 링크 등을 포함한 문서를 작성하는 셀입니다.
# 제목 ## 부제목 - 목록 항목 1 - 목록 항목 2 **굵은 글씨**와 *기울임꼴*

기본 작업

새 노트북 생성: 시작 페이지에서 [New] → [Python 3] 선택
노트북 업로드: [Upload] 버튼을 클릭하여 .ipynb 파일 업로드
노트북 저장: Ctrl+S 또는 메뉴에서 [File] → [Save and Checkpoint]
노트북 내보내기: [File] → [Download as] → 원하는 형식 선택

마크다운 활용법

제목: # 대제목, ## 중제목, ### 소제목
목록: * 항목 또는 - 항목 (순서가 없는 목록), 1. 항목 (순서가 있는 목록)
강조: **굵은 글씨** 또는 *기울임꼴*
코드: `인라인 코드`, ```python 여러 줄 코드 ```
링크: [링크 텍스트](URL)
이미지: ![대체 텍스트](이미지 URL)
표: | 열1 | 열2 | 와 같은 형식으로 작성
수식: $$ E = mc^2 $$ 형식으로 LaTeX 수식 작성

효율적인 사용 팁

자동 완성: 코드 작성 중 Tab 키를 누르면 자동 완성 제안
도움말: 함수 뒤에 ? 또는 ?? 입력 후 실행 (예: print?)
커널 관리: 오류 발생 시 [Kernel] → [Restart] 실행
체크포인트: 중요 지점에서 주기적으로 저장하여 작업 보호
Tip: 주피터 노트북의 단축키를 이용하면 작업 효율이 크게 높아집니다. 명령 모드(파란색 테두리)와 편집 모드(녹색 테두리)의 단축키가 다르니 주의하세요. Esc 키를 눌러 명령 모드로, Enter 키를 눌러 편집 모드로 전환할 수 있습니다. 단축키 목록을 보려면 [Help] → [Keyboard Shortcuts]를 클릭하세요.
