조건문 (Conditional Statements)

조건문은 프로그램의 흐름을 제어하는 핵심 구문으로, 업무 자동화에서 의사결정 로직을 구현하는 데 필수적입니다.

기본 조건문 구조

파이썬의 조건문은 if, elif, else 키워드를 사용하여 특정 조건이 참일 때 코드 블록을 실행합니다.

조건 검사
→
True: 코드 실행
→
다음 코드
기본 if-else 조건문 예제 코드를 작성해줘
# 기본 if 구문
점수 = 85
if 점수 >= 90:
print("A 학점입니다.")
elif 점수 >= 80:
print("B 학점입니다.")
elif 점수 >= 70:
print("C 학점입니다.")
else:
print("F 학점입니다.")
출력 결과:
B 학점입니다.
들여쓰기(Indentation) 주의!

파이썬에서는 들여쓰기가 코드 블록을 구분하는 문법적 요소입니다. 일관된 들여쓰기를 유지해야 합니다 (보통 4칸 공백 사용).

중첩 조건문

조건문 안에 또 다른 조건문을 중첩해서 사용할 수 있습니다. 복잡한 의사결정 로직을 구현할 때 유용합니다.

중첩된 if문 예제를 보여줘
나이 = 25
회원_등급 = "골드"
if 나이 >= 20:
print("성인입니다.")
if 회원_등급 == "골드":
print("골드 회원 할인이 적용됩니다.")
elif 회원_등급 == "실버":
print("실버 회원 할인이 적용됩니다.")
else:
print("할인이 적용되지 않습니다.")
else:
print("미성년자입니다.")
출력 결과:
성인입니다.
골드 회원 할인이 적용됩니다.
논리 연산자

여러 조건을 조합하기 위해 and, or, not 논리 연산자를 사용합니다.

논리 연산자를 사용한 조건문 예제 코드를 보여줘
나이 = 30
소득 = 3500000
신용점수 = 750
# AND 연산자 - 모든 조건이 참이어야 함
if 나이 >= 25 and 소득 > 3000000:
print("대출 기본 조건을 충족합니다.")
# OR 연산자 - 하나라도 참이면 참
if 신용점수 > 800 or 소득 > 5000000:
print("우대 금리를 적용합니다.")
else:
print("일반 금리가 적용됩니다.")
# NOT 연산자 - 조건의 결과를 반전
if not (나이 < 20):
print("성인입니다.")
진릿값 평가

파이썬에서는 0, None, ""(빈 문자열), [](빈 리스트) 등은 False로 평가되고, 그 외의 값들은 True로 평가됩니다.

데이터프레임에서의 조건문 활용

데이터프레임 처리 자동화에서 조건문은 데이터 필터링이나 값 변환에 중요하게 사용됩니다.

판다스 데이터프레임에서 조건문을 활용한 코드 예시
import pandas as pd
직원_df = pd.DataFrame({
'이름': ['김철수', '이영희', '박지민', '최동욱'],
'부서': ['영업', '개발', '개발', '영업'],
'실적': [90, 85, 92, 78]
})
# 조건부 컬럼 추가
직원_df['성과급'] = '없음'
# 조건에 따라 값 할당
for i in range(len(직원_df)):
if 직원_df.loc[i, '실적'] >= 90:
직원_df.loc[i, '성과급'] = '상'
elif 직원_df.loc[i, '실적'] >= 80:
직원_df.loc[i, '성과급'] = '중'
else:
직원_df.loc[i, '성과급'] = '하'
결과 데이터프레임:
  이름  부서 실적 성과급
0 김철수 영업  90   상
1 이영희 개발  85   중
2 박지민 개발  92   상
3 최동욱 영업  78   하
"조건문은 프로그램에 '지능'을 부여하는 첫 번째 단계입니다"
