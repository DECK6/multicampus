BeautifulSoup4

HTML/XML 파싱 라이브러리로, 웹 페이지에서 데이터를 추출하는데 사용됩니다.

pip install beautifulsoup4

WordCloud

텍스트 데이터를 시각적으로 표현하는 워드클라우드를 생성합니다.

pip install wordcloud

python-pptx

파워포인트 파일을 생성하고 편집할 수 있는 라이브러리입니다.

pip install python-pptx

smtplib

파이썬 기본 내장 라이브러리로, 이메일 전송 기능을 제공합니다.

openpyxl

엑셀 파일을 생성하고 편집할 수 있는 라이브러리입니다.

pip install openpyxl

konlpy

한글 자연어 처리를 위한 형태소 분석 라이브러리입니다.

pip install konlpy
