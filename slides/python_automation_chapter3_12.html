웹 자동화와 API 활용

파이썬으로 웹 데이터를 수집하고 처리하는 자동화 방법을 알아봅니다. 웹 스크래핑과 API 연동을 통해 다양한 데이터 소스를 자동으로 활용할 수 있습니다.

웹 자동화 라이브러리

파이썬에는 웹 페이지 데이터를 수집하고 처리하기 위한 다양한 라이브러리가 있습니다.

Requests
HTTP 요청을 쉽게 보낼 수 있는 라이브러리로, 웹 페이지 가져오기, API 호출 등에 사용됩니다. 직관적인 인터페이스로 웹 자동화의 기본이 됩니다.
BeautifulSoup
HTML 및 XML 파일에서 데이터를 추출하는 라이브러리로, 웹 스크래핑에 필수적입니다. 웹 페이지의 구조를 파싱하여 원하는 정보를 쉽게 찾을 수 있습니다.
Selenium
브라우저를 직접 제어할 수 있는 라이브러리로, 동적 웹 페이지 처리, 폼 작성, 버튼 클릭 등 사용자 상호작용이 필요한 경우에 사용됩니다.

웹 스크래핑 주의사항

웹 스크래핑 시 다음 사항을 항상 고려해야 합니다:

  • API가 제공되는 경우 API 우선 사용하기
웹 스크래핑 기본 예제

Requests와 BeautifulSoup을 활용한 기본 웹 스크래핑 예제입니다.

파이썬으로 웹페이지에서 뉴스 헤드라인을 수집하는 코드를 작성해줘
import requests
from bs4 import BeautifulSoup
import pandas as pd
import datetime
# 뉴스 사이트 URL (예시)
url = "https://news.naver.com/main/main.naver?mode=LSD&mid=shm&sid1=101"
# User-Agent 설정 (필수 - 접근 제한 방지)
headers = {
"User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
}
# 웹 페이지 요청
response = requests.get(url, headers=headers)
# 응답 확인
if response.status_code == 200:
# HTML 파싱
soup = BeautifulSoup(response.text, 'html.parser')
# 헤드라인 뉴스 선택 (가상의 선택자, 실제 사이트에 맞게 수정 필요)
headlines = soup.select('.sh_text_headline')
summaries = soup.select('.sh_text_lede')
# 데이터 저장용 리스트
news_data = []
# 최대 10개 뉴스 추출
for i in range(min(10, len(headlines))):
news_data.append({
'제목': headlines[i].text.strip(),
'요약': summaries[i].text.strip() if i < len(summaries) else "-",
'수집일시': datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
})
print(f"수집: {headlines[i].text.strip()}")
# 데이터프레임 변환 및 엑셀 저장
df = pd.DataFrame(news_data)
df.to_excel(f"경제뉴스_{datetime.datetime.now().strftime('%Y%m%d')}.xlsx", index=False)
print(f"총 {len(news_data)}개의 뉴스 헤드라인을 저장했습니다.")
else:
print(f"오류 발생: 상태 코드 {response.status_code}")
API 활용 기본 데이터 연동

API(Application Programming Interface)는 다른 서비스의 데이터나 기능을 쉽게 활용할 수 있게 해주는 인터페이스입니다.

공공 API를 사용해 날씨 정보를 가져오는 파이썬 코드를 작성해줘
import requests
import pandas as pd
import json
from datetime import datetime
# OpenWeatherMap API 예제 (실제 사용 시 API 키 필요)
API_KEY = "your_api_key_here"
city = "Seoul"
# API 엔드포인트 URL
url = f"http://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}&units=metric"
# API 요청
response = requests.get(url)
if response.status_code == 200:
# JSON 응답 파싱
weather_data = response.json()
# 필요한 데이터 추출
weather = {
'도시': weather_data['name'],
'날씨': weather_data['weather'][0]['description'],
'현재기온': weather_data['main']['temp'],
'체감기온': weather_data['main']['feels_like'],
'습도': weather_data['main']['humidity'],
'풍속': weather_data['wind']['speed'],
'수집시간': datetime.now().strftime("%Y-%m-%d %H:%M:%S")
}
print(f"{weather['도시']}의 현재 날씨: {weather['날씨']}")
print(f"현재기온: {weather['현재기온']}°C, 체감기온: {weather['체감기온']}°C")
# 데이터프레임으로 변환
df = pd.DataFrame([weather])
# 엑셀로 저장 (기존 파일에 추가)
try:
기존_데이터 = pd.read_excel('날씨_데이터.xlsx')
df = pd.concat([기존_데이터, df], ignore_index=True)
except FileNotFoundError:
pass # 파일이 없으면 새로 생성
df.to_excel('날씨_데이터.xlsx', index=False)
else:
print(f"오류: {response.status_code}, {response.text}")

셀레니움을 활용한 자동화

Selenium은 웹 브라우저를 프로그래밍으로 제어하여 복잡한 웹 자동화 작업을 수행할 수 있습니다.

셀레니움으로 웹사이트에 로그인하고 데이터를 수집하는 코드를 작성해줘
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
import time
import pandas as pd
# 웹드라이버 설정 (Chrome 드라이버 필요)
driver_path = "./chromedriver" # 크롬 드라이버 경로
service = Service(driver_path)
driver = webdriver.Chrome(service=service)
# 웹사이트 접속
url = "https://example.com/login" # 로그인 페이지 URL
driver.get(url)
try:
# 로그인 (예시)
username = WebDriverWait(driver, 10).until(
EC.presence_of_element_located((By.ID, "username"))
)
password = driver.find_element(By.ID, "password")
username.send_keys("your_username")
password.send_keys("your_password")
login_button = driver.find_element(By.XPATH, "//button[contains(text(), '로그인')]")
login_button.click()
# 로그인 후 대시보드 페이지로 이동 대기
WebDriverWait(driver, 10).until(
EC.presence_of_element_located((By.CLASS_NAME, "dashboard"))
)
# 데이터가 있는 페이지로 이동
driver.get("https://example.com/reports")
# 테이블 데이터 수집
table_rows = WebDriverWait(driver, 10).until(
EC.presence_of_all_elements_located((By.XPATH, "//table/tbody/tr"))
)
data = []
for row in table_rows:
cols = row.find_elements(By.TAG_NAME, "td")
row_data = [col.text for col in cols]
data.append(row_data)
# 데이터프레임으로 변환
columns = ['항목1', '항목2', '항목3', '항목4'] # 실제 컬럼명으로 수정
df = pd.DataFrame(data, columns=columns)
df.to_excel("웹자동화_수집데이터.xlsx", index=False)
print(f"총 {len(data)}개 행의 데이터를 수집했습니다.")
finally:
driver.quit() # 브라우저 종료
웹 자동화 활용 사례

웹 자동화 기술은 다음과 같은 업무에 활용될 수 있습니다:

  • 정기적인 보고서 데이터 자동 수집
  • 온라인 양식 자동 제출 및 처리
  • 경쟁사 가격/제품 정보 모니터링
  • 소셜 미디어 데이터 분석
  • 웹사이트 테스트 자동화
