데이터프레임 선택과 필터링

데이터프레임에서 특정 데이터를 선택하고 필터링하는 방법을 알아봅니다. 이는 데이터 처리 자동화의 핵심 작업입니다.

열(컬럼) 선택하기

특정 열 또는 여러 열을 선택하여 필요한 데이터만 추출할 수 있습니다.

판다스 데이터프레임에서 특정 열을 선택하는 방법을 알려줘
import pandas as pd
# 샘플 데이터프레임
df = pd.DataFrame({
'이름': ['홍길동', '김철수', '이영희'],
'나이': [25, 30, 28],
'부서': ['개발', '마케팅', '인사'],
'급여': [3000, 3500, 3200]
})
# 단일 열 선택 (Series 반환)
이름_열 = df['이름']
# 여러 열 선택 (DataFrame 반환)
이름_부서 = df[['이름', '부서']]
여러 열 선택 출력 결과:
  이름 부서
0 홍길동 개발
1 김철수 마케팅
2 이영희 인사
행(인덱스) 선택하기

인덱스 기반으로 행을 선택하거나 특정 범위의 행을 선택할 수 있습니다.

데이터프레임에서 특정 행을 인덱스로 선택하는 코드 예시
# loc - 라벨 기반 인덱싱
첫번째_행 = df.loc[0] # 첫 번째 행 (Series로 반환)
# iloc - 위치 기반 인덱싱
첫번째_행 = df.iloc[0] # 첫 번째 행
여러_행 = df.iloc[0:2] # 첫 번째와 두 번째 행
loc vs iloc

loc: 라벨(인덱스 이름)을 기준으로 선택

iloc: 위치(0부터 시작하는 정수)를 기준으로 선택

조건으로 데이터 필터링하기

불리언(Boolean) 마스킹을 사용하여 특정 조건을 만족하는 데이터만 필터링할 수 있습니다.

판다스에서 조건을 사용해 데이터를 필터링하는 방법을 보여줘
# 단일 조건 필터링
개발자_직원 = df[df['부서'] == '개발']
# 나이가 28세 이상인 직원
나이_많은_직원 = df[df['나이'] >= 28]
# 복합 조건 (AND)
조건1 = df['부서'] == '개발'
조건2 = df['급여'] > 3000
고급여_개발자 = df[조건1 & 조건2]
# 복합 조건 (OR)
마케팅_인사 = df[(df['부서'] == '마케팅') | (df['부서'] == '인사')]
조건 필터링 출력 결과 (개발자_직원):
  이름 나이 부서 급여
0 홍길동 25 개발 3000
행과 열 동시에 선택하기

특정 행과 열을 동시에 선택하여 필요한 데이터 하위 집합을 추출할 수 있습니다.

데이터프레임에서 특정 행과 열을 동시에 선택하는 방법을 알려줘
# loc을 사용한 행과 열 동시 선택
subset1 = df.loc[0:1, ['이름', '급여']]
# iloc을 사용한 행과 열 동시 선택
subset2 = df.iloc[0:2, [0, 3]] # 첫 2행, 첫 번째와 네 번째 열
# 조건 필터링과 열 선택 조합
개발자_이름_급여 = df[df['부서'] == '개발'][['이름', '급여']]
업무 자동화 팁

데이터 필터링은 보고서 자동화, 데이터 분석, 특정 조건의 레코드 찾기 등 다양한 업무 자동화에 필수적인 기술입니다. 엑셀의 필터 기능과 유사하지만 더 강력하고 유연합니다.

"데이터프레임의 선택과 필터링은 모든 데이터 처리 자동화의 기본이자 시작점입니다"
