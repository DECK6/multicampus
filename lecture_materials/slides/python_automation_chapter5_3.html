키보드 자동화

PyAutoGUI를 사용하면 키보드 입력을 자동화하여 텍스트 입력, 특수 키 조작, 단축키(핫키) 실행 등을 프로그래밍 방식으로 제어할 수 있습니다. 키보드 관련 함수들은 현재 키보드 커서가 있는 위치에 적용됩니다.

텍스트 입력

텍스트를 자동으로 타이핑하는 함수입니다. 글자와 글자 사이에 지연 시간을 설정할 수 있습니다.

import pyautogui

# 텍스트 입력 (즉시 입력)
pyautogui.write('Hello World!')

# 글자 간 0.25초 간격으로 천천히 타이핑
pyautogui.write('Hello World!', interval=0.25)

# 엔터키 입력 추가
pyautogui.write('Hello World!\n')

특수 키 입력

Enter, Esc, 화살표 키 등 특수 키를 입력하는 함수입니다. 키 목록은 KEYBOARD_KEYS 상수에 정의되어 있습니다.

# 특수 키 입력 (Enter 키)
pyautogui.press('enter')

# 화살표 키 입력
pyautogui.press('left')  # 왼쪽 화살표
pyautogui.press('right')  # 오른쪽 화살표
pyautogui.press('up')  # 위쪽 화살표
pyautogui.press('down')  # 아래쪽 화살표

# 여러 키 순차적으로 입력
pyautogui.press(['left', 'left', 'left'])

# 동일한 키 여러 번 입력
pyautogui.press('left', presses=3, interval=0.2)

키보드 단축키 (핫키)

Ctrl+C, Alt+Tab과 같은 키보드 단축키를 실행할 수 있는 함수입니다.

# Ctrl+C (복사)
pyautogui.hotkey('ctrl', 'c')

# Ctrl+V (붙여넣기)
pyautogui.hotkey('ctrl', 'v')

# Alt+Tab (창 전환)
pyautogui.hotkey('alt', 'tab')

# Ctrl+Alt+Delete
pyautogui.hotkey('ctrl', 'alt', 'delete')

# Ctrl+Shift+Esc (작업 관리자)
pyautogui.hotkey('ctrl', 'shift', 'esc')

키 개별 제어

키를 누르고 떼는 동작을 개별적으로 제어할 수 있는 함수입니다. 특정 키를 오래 누르거나 hold 컨텍스트 매니저를 사용할 수 있습니다.

# 키 누르기
pyautogui.keyDown('shift')

# 다른 키 입력
pyautogui.press('left')  # Shift+왼쪽 화살표 효과

# 키 떼기
pyautogui.keyUp('shift')

# hold() 컨텍스트 매니저 사용 (Python 3.6+)
with pyautogui.hold('shift'):
    pyautogui.press(['left', 'left', 'left'])
    # 이 블록이 끝나면 자동으로 shift 키가 떼짐
유용한 팁

문자열 내에서 특수 키를 사용하려면 write() 대신 press()나 pyautogui.typewrite()를 사용하세요. 예를 들어, 텍스트를 입력한 후 Tab 키를 누르고 다른 텍스트를 입력하려면: pyautogui.write('사용자명') 후 pyautogui.press('tab') 그리고 pyautogui.write('비밀번호')와 같이 조합해서 사용할 수 있습니다.
