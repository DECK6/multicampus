메신저/문자/메일 자동화
이메일, 문자 메시지, 메신저 등을 통한 자동 알림 및 메시지 전송 시스템을 구축하여 알림, 보고서 전송, 마케팅 메시지 등을 자동화할 수 있습니다. 파이썬의 여러 라이브러리를 활용하면 다양한 통신 채널을 통합적으로 관리할 수 있습니다.

주요 라이브러리

SMTPLIB
파이썬 표준 라이브러리로, SMTP(Simple Mail Transfer Protocol) 서버를 통해 이메일을 전송할 수 있습니다. 텍스트 메일부터 HTML 형식, 첨부 파일이 있는 메일까지 다양한 형태의 이메일을 보낼 수 있습니다.
win32com.client
Windows COM(Component Object Model) 인터페이스를 통해 Microsoft Office 제품과 통합할 수 있는 라이브러리입니다. Outlook을 통한 이메일 전송, 자동 응답, 메일 읽기 등 다양한 이메일 자동화가 가능합니다.
Twilio
클라우드 기반 통신 플랫폼으로, SMS, 음성 통화, WhatsApp 등을 통해 메시지를 보낼 수 있습니다. 간단한 API를 통해 전 세계 어디서나 문자 메시지를 전송할 수 있어 알림 시스템에 적합합니다.

활용 사례

자동 알림 시스템
재고 부족, 시스템 오류, 중요 이벤트 발생 시 자동 알림 전송
보고서 자동 배포
일/주/월간 보고서를 자동으로 생성하여 관련자에게 이메일 전송
일정 알림
회의, 마감일, 중요 이벤트 등에 대한 알림 메시지 자동 전송
고객 커뮤니케이션
견적서 발송, 주문 확인, 배송 알림 등 고객 소통 자동화
라이브러리 장점 단점 적합한 상황
SMTPLIB 표준 라이브러리, 설치 불필요 기본 기능만 제공 간단한 이메일 자동화
win32com Outlook 기능 완벽 활용 Windows 환경에서만 작동 Outlook 통합 필요 시
Twilio 글로벌 SMS 지원, 안정적 유료 서비스 SMS 알림 필요 시

# SMTPLIB를 활용한 이메일 전송 예제 import smtplib from email.mime.multipart import MIMEMultipart from email.mime.text import MIMEText from email.mime.application import MIMEApplication import os # 이메일 설정 sender_email = "[email protected]" receiver_email = "[email protected]" password = "your_password" # 앱 비밀번호 권장 # 메일 객체 생성 msg = MIMEMultipart() msg['Subject'] = '월간 보고서 - 6월' msg['From'] = sender_email msg['To'] = receiver_email # HTML 본문 작성 html = '''

6월 월간 보고서

안녕하세요,

6월 월간 보고서를 첨부합니다.

주요 내용:

  • 매출: 전월 대비 15% 상승
  • 신규 고객: 23명 유입
  • 주요 프로젝트 진행 상황: 85%

자세한 내용은 첨부 파일을 확인해주세요.

감사합니다.

''' # HTML 본문 추가 msg.attach(MIMEText(html, 'html')) # 첨부 파일 추가 file_path = "월간보고서_6월.xlsx" with open(file_path, 'rb') as file: attachment = MIMEApplication(file.read(), _subtype="xlsx") attachment.add_header( 'Content-Disposition', 'attachment', filename=os.path.basename(file_path) ) msg.attach(attachment) # 메일 서버 연결 및 전송 try: server = smtplib.SMTP_SSL('smtp.gmail.com', 465) server.login(sender_email, password) server.sendmail(sender_email, receiver_email, msg.as_string()) print("이메일이 성공적으로 전송되었습니다!") except Exception as e: print(f"이메일 전송 중 오류 발생: {e}") finally: server.quit()

# win32com을 활용한 Outlook 이메일 전송 예제 import win32com.client from datetime import datetime # Outlook 애플리케이션 연결 outlook = win32com.client.Dispatch('Outlook.Application') namespace = outlook.GetNamespace('MAPI') # 새 메일 아이템 생성 mail = outlook.CreateItem(0) # 0은 olMailItem # 수신자 설정 mail.To = '[email protected]' mail.CC = '[email protected]' # 메일 제목 설정 today = datetime.now().strftime("%Y-%m-%d") mail.Subject = f'일일 업무 보고 - {today}' # HTML 본문 작성 mail.HTMLBody = '''

일일 업무 보고

안녕하세요,

오늘의 주요 업무 내용을 공유드립니다.

완료된 작업

  • 고객 데이터 분석 완료
  • 프로젝트 A 문서화 작업

진행 중인 작업

  • 시스템 업데이트 (70% 완료)
  • 신규 기능 테스트

자세한 내용은 첨부 파일을 참고해 주세요.

감사합니다.

''' # 첨부 파일 추가 attachment_path = r'C:\Reports\Daily_Report.xlsx' mail.Attachments.Add(attachment_path) # 중요도 설정 (1: 낮음, 2: 보통, 3: 높음) mail.Importance = 2 # 메일 전송 try: mail.Send() print("이메일이 성공적으로 전송되었습니다!") except Exception as e: print(f"이메일 전송 중 오류 발생: {e}")
Tip: Gmail을 사용하여 이메일을 전송할 때는 보안을 위해 앱 비밀번호를 사용하는 것이 좋습니다. 구글 계정 설정 → 보안 → 2단계 인증 → 앱 비밀번호에서 생성할 수 있습니다. 또한 민감한 정보(비밀번호 등)는 환경 변수나 별도의 설정 파일에 저장하여 코드와 분리하는 것이 보안에 좋습니다.
