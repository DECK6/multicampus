smtplib 라이브러리 소개

파이썬의 smtplib는 SMTP(Simple Mail Transfer Protocol) 프로토콜을 사용하여 이메일을 보내기 위한 내장 라이브러리입니다.

ChatGPT Prompt: "Write basic Python code to import and setup smtplib" 
import smtplib
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart

# SMTP 서버 설정
smtp_server = "smtp.gmail.com"
smtp_port = 587

# SMTP 객체 생성
smtp = smtplib.SMTP(smtp_server, smtp_port)
smtp.starttls()  # TLS 보안 시작

보안 기능

TLS/SSL 암호화를 통해 안전한 이메일 전송을 보장합니다. starttls() 메서드로 보안 연결을 시작합니다.

다양한 이메일 형식

텍스트, HTML, 첨부파일 등 다양한 형식의 이메일을 지원합니다. MIME 타입을 통해 복합 메시지 구성이 가능합니다.

여러 SMTP 서버 지원

Gmail, Outlook 등 다양한 이메일 서비스의 SMTP 서버와 호환됩니다. 포트 설정으로 유연한 연결이 가능합니다.

간단한 API

sendmail(), login() 등 직관적인 메서드를 제공하여 쉽게 이메일 전송 자동화를 구현할 수 있습니다.
