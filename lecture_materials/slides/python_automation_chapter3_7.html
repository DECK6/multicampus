반복문 - while 루프와 제어문

특정 조건이 만족하는 동안 반복해서 실행되는 while 루프와 반복문의 흐름을 제어하는 명령어들을 알아봅니다.

while 루프 기본 조건 기반

while 루프는 특정 조건이 참인 동안 코드 블록을 계속 실행합니다. 조건이 거짓이 되면 반복 실행을 멈춥니다.

시작
조건 검사
True
False
코드 블록 실행
종료
파이썬에서 while 루프 기본 사용법 예시를 보여줘
# 카운트다운 예제
카운트 = 5
while 카운트 > 0:
print(f"{카운트}...")
카운트 -= 1
print("발사!")
출력 결과:
5...
4...
3...
2...
1...
발사!
무한 루프 주의!

while 루프에서는 반드시 조건이 언젠가 False가 되도록 설계해야 합니다. 그렇지 않으면 프로그램이 영원히 실행되는 무한 루프에 빠질 수 있습니다.

break와 continue 문

break와 continue를 사용하여 반복문 실행을 제어할 수 있습니다.

파이썬에서 break와 continue 사용 예시 코드를 보여줘
# break 예제 - 특정 조건에서 반복 종료
숫자_목록 = [1, 3, 5, 7, 9, 11, 13, 15]
for 숫자 in 숫자_목록:
print(f"현재 숫자: {숫자}")
if 숫자 > 10:
print("10보다 큰 숫자를 찾았습니다. 반복을 종료합니다.")
break
# continue 예제 - 특정 조건에서 다음 반복으로 건너뛰기
for i in range(1, 11):
if i % 2 == 0: # 짝수면 건너뛰기
continue
print(f"홀수: {i}")
break 출력 결과:
현재 숫자: 1
현재 숫자: 3
현재 숫자: 5
현재 숫자: 7
현재 숫자: 9
현재 숫자: 11
10보다 큰 숫자를 찾았습니다. 반복을 종료합니다.
중첩 반복문

반복문 안에 또 다른 반복문을 중첩하여 사용할 수 있습니다. 다차원 데이터 처리나 복잡한 패턴 생성에 유용합니다.

중첩 for 루프를 사용한 간단한 예제 코드를 보여줘
# 구구단 출력 (2단부터 5단까지)
for i in range(2, 6): # 외부 루프: 단수
print(f"--- {i}단 ---")
for j in range(1, 10): # 내부 루프: 곱하는 수
print(f"{i} x {j} = {i*j}")
print("") # 단 사이 빈 줄
2차원 리스트를 처리하는 중첩 반복문 예시를 보여줘
# 2차원 리스트 처리
성적_데이터 = [
[85, 90, 78], # 국어, 영어, 수학
[92, 88, 95],
[78, 85, 83]
]
for i, 학생_성적 in enumerate(성적_데이터):
총점 = 0
print(f"학생 {i+1}의 성적:")
for j, 점수 in enumerate(['국어', '영어', '수학']):
print(f"{점수}: {학생_성적[j]}점")
총점 += 학생_성적[j]
평균 = 총점 / len(학생_성적)
print(f"평균: {평균:.1f}점\n")
업무 자동화에서의 응용

반복문은 업무 자동화의 핵심 요소로, 다양한 실무 상황에서 활용할 수 있습니다.

파일 처리 자동화에 while 루프와 조건문을 활용하는 예시 코드를 보여줘
import os
# 폴더 내 모든 CSV 파일 처리 자동화
폴더_경로 = "./데이터파일"
파일_목록 = os.listdir(폴더_경로)
처리_카운트 = 0
for 파일명 in 파일_목록:
if 파일명.endswith(".csv"):
파일_경로 = os.path.join(폴더_경로, 파일명)
print(f"처리 중: {파일명}")
# 파일 처리 로직...
처리_카운트 += 1
if 처리_카운트 >= 10:
print("최대 처리 개수에 도달했습니다.")
break
print(f"총 {처리_카운트}개의 CSV 파일을 처리했습니다.")
실무 활용 사례

반복문은 다음과 같은 자동화 작업에 활용할 수 있습니다:

  • 다수의 파일 일괄 처리 (이름 변경, 내용 수정)
  • 대량 이메일 발송 자동화
  • 주기적인 데이터 수집 및 모니터링
  • 보고서 자동 생성 및 배포
  • 대용량 데이터 분석 작업 분할 처리
