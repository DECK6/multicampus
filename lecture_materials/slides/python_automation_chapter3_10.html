데이터 시각화 기초

파이썬을 활용한 데이터 시각화의 기초를 배우고, 업무 자동화 과정에서 데이터를 효과적으로 표현하는 방법을 알아봅니다.

데이터 시각화 라이브러리

파이썬에는 강력한 데이터 시각화 라이브러리들이 있으며, 이를 활용하여 다양한 차트와 그래프를 생성할 수 있습니다.

Matplotlib
파이썬의 기본 시각화 라이브러리로, 다양한 차트와 그래프를 생성할 수 있습니다. 대부분의 시각화 라이브러리는 Matplotlib을 기반으로 합니다.
Seaborn
Matplotlib 기반의 고급 시각화 라이브러리로, 통계 데이터 시각화에 최적화되어 있으며 더 아름다운 기본 스타일을 제공합니다.
Plotly
인터랙티브한 시각화를 위한 라이브러리로, 차트에 마우스를 올리면 상세 정보를 볼 수 있고 확대/축소도 가능합니다.
Pandas 내장 시각화
판다스 데이터프레임에 내장된 plot() 함수를 사용하여 간단하게 데이터 시각화를 생성할 수 있습니다.
시각화 선택 가이드

적절한 시각화 유형을 선택하는 것이 중요합니다:

  • 막대 차트: 범주별 비교에 적합
  • 선 그래프: 시간에 따른 추세 분석에 적합
  • 파이 차트: 전체 대비 비율 표시에 적합
  • 히스토그램: 분포 확인에 적합
  • 산점도: 두 변수 간의 관계 분석에 적합
Matplotlib 기본 사용법
Matplotlib을 사용하여 간단한 막대 그래프를 그리는 코드를 작성해줘
import matplotlib.pyplot as plt
import numpy as np
# 데이터 준비
부서 = ['영업', '마케팅', '개발', '인사', '재무']
실적 = [85, 67, 92, 55, 70]
# 한글 폰트 설정 (Windows 기준)
plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False
# 그래프 생성
plt.figure(figsize=(10, 6))
plt.bar(부서, 실적, color='#4B8BBE')
# 그래프 제목 및 레이블 설정
plt.title('부서별 분기 실적', fontsize=15)
plt.xlabel('부서')
plt.ylabel('실적 (단위: 백만원)')
# 데이터 레이블 추가
for i, v in enumerate(실적):
plt.text(i, v+2, str(v), ha='center')
plt.tight_layout()
plt.savefig('부서별_실적.png')
plt.show()
부서별 분기 실적 그래프 (막대 차트 이미지)
판다스로 데이터 시각화하기

판다스 데이터프레임은 간편한 시각화 기능을 내장하고 있어 빠르게 차트를 생성할 수 있습니다.

판다스 데이터프레임을 사용해서 판매 데이터를 시각화하는 코드를 작성해줘
import pandas as pd
import matplotlib.pyplot as plt
# 샘플 데이터 생성
날짜 = pd.date_range(start='2023-01-01', periods=6, freq='M')
판매_데이터 = pd.DataFrame({
'날짜': 날짜,
'제품A': [120, 135, 128, 145, 160, 152],
'제품B': [85, 90, 88, 95, 102, 98]
})
# 날짜를 인덱스로 설정
판매_데이터.set_index('날짜', inplace=True)
# 한글 폰트 설정
plt.rcParams['font.family'] = 'Malgun Gothic'
# 판다스의 plot 메서드로 선 그래프 생성
판매_데이터.plot(figsize=(10, 5),
title='월별 제품 판매량 추이',
grid=True,
style=['-o', '-s'],
colormap='viridis')
plt.ylabel('판매량 (단위: 개)')
plt.legend(title='제품')
plt.tight_layout()
plt.savefig('월별_판매추이.png')
plt.show()
월별 제품 판매량 추이 그래프 (선 그래프 이미지)
시각화 자동화 활용 보고서 작성

데이터 시각화를 자동화하여 정기 보고서 생성, 데이터 모니터링, 의사결정 지원 등에 활용할 수 있습니다.

여러 데이터 파일을 읽어서 자동으로 그래프를 생성하는 코드를 작성해줘
import pandas as pd
import matplotlib.pyplot as plt
import glob
import os
# 그래프 저장 폴더 생성
그래프_폴더 = "./보고서/그래프"
os.makedirs(그래프_폴더, exist_ok=True)
# 모든 CSV 파일 경로 가져오기
csv_파일목록 = glob.glob("./데이터/*.csv")
for 파일경로 in csv_파일목록:
파일명 = os.path.basename(파일경로).split('.')[0]
print(f"{파일명} 처리 중...")
try:
# CSV 파일 읽기
df = pd.read_csv(파일경로)
# 그래프 생성 (예: 막대 그래프)
plt.figure(figsize=(10, 6))
plt.title(f"{파일명} 데이터 분석", fontsize=15)
# 데이터 유형에 따른 차트 선택
if '날짜' in df.columns:
df.set_index('날짜').plot(ax=plt.gca())
plt.xticks(rotation=45)
elif '카테고리' in df.columns:
df.groupby('카테고리').sum().plot(kind='bar', ax=plt.gca())
else:
df.plot(kind='bar', ax=plt.gca())
plt.tight_layout()
# 그래프 저장
그래프_경로 = os.path.join(그래프_폴더, f"{파일명}_차트.png")
plt.savefig(그래프_경로)
plt.close()
print(f" 그래프 저장 완료: {그래프_경로}")
except Exception as e:
print(f" 오류 발생: {e}")
자동화 활용 시나리오

데이터 시각화 자동화는 다음과 같은 업무에 유용합니다:

  • 일일/주간/월간 판매 보고서 자동 생성
  • 여러 지점/부서의 실적 비교 차트 생성
  • KPI 대시보드 업데이트 자동화
  • 데이터 이상치 시각적 감지 시스템
