ChatGPT를 활용한 코드 개선 및 타언어 변환 방법

ChatGPT는 기존 코드의 품질을 향상시키거나 다른 프로그래밍 언어로 변환하는 작업에도 유용합니다. 이를 통해 코드 최적화 기법을 학습하고 다양한 언어의 특성을 이해할 수 있습니다.

코드 개선 및 최적화
ChatGPT 프롬프트
다음 파이썬 코드를 더 효율적으로 개선해줘. 가독성과 성능을 모두 고려해서 최적화해줘: for i in range(1, 101): if i % 3 == 0 and i % 5 == 0: print("FizzBuzz") elif i % 3 == 0: print("Fizz") elif i % 5 == 0: print("Buzz") else: print(i)
개선 전
# 원래 코드 for i in range(1, 101): if i % 3 == 0 and i % 5 == 0: print("FizzBuzz") elif i % 3 == 0: print("Fizz") elif i % 5 == 0: print("Buzz") else: print(i)
개선 후
# 최적화된 코드 def fizzbuzz(n): for i in range(1, n+1): result = "" if i % 3 == 0: result += "Fizz" if i % 5 == 0: result += "Buzz" print(result or i) fizzbuzz(100)
타언어 변환
ChatGPT 프롬프트
다음 자바스크립트 코드를 파이썬으로 변환해줘: function fibonacci(n) { if (n <= 1) return n; return fibonacci(n-1) + fibonacci(n-2); } const result = fibonacci(10); console.log(`The 10th Fibonacci number is: ${result}`);
JavaScript
function fibonacci(n) { if (n <= 1) return n; return fibonacci(n-1) + fibonacci(n-2); } const result = fibonacci(10); console.log(`The 10th Fibonacci number is: ${result}`);
Python
def fibonacci(n): if n <= 1: return n return fibonacci(n-1) + fibonacci(n-2) result = fibonacci(10) print(f"The 10th Fibonacci number is: {result}")
