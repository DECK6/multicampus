샘플 제목

안녕하세요 [이름]님,

이것은 이메일 본문 메시지 예시입니다. 템플릿이 어떻게 보일지 보여줍니다.

클릭하세요

디자인 요소

  • 반응형 디자인 (max-width 사용)
  • 안전한 웹폰트 사용
  • 적절한 여백과 간격
  • 클릭 유도 버튼 디자인

템플릿 변수

  • {title}: 이메일 제목
  • {name}: 수신자 이름
  • {message}: 본문 내용
  • {cta_link}, {cta_text}: 버튼 링크와 텍스트

이메일 클라이언트 호환성 팁

  • 인라인 스타일 사용 권장
  • 테이블 기반 레이아웃 고려
  • 이미지에 alt 텍스트 필수 추가
  • 외부 리소스 사용 최소화
