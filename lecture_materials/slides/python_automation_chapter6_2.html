HTML 기초 이해하기
HTML 문서의 기본 구조
HTML <!DOCTYPE html>
<html>
<head>
<title>웹페이지 제목</title>
</head>
<body>
<h1>제목입니다</h1>
<p>단락입니다</p>
<a href="https://example.com">링크</a>
</body>
</html>
HTML 태그의 구조
<태그명 속성명="속성값">내용</태그명>

• 태그(Tag): HTML 요소의 시작과 끝을 나타냄
• 속성(Attribute): 요소의 추가 정보 제공
• 내용(Content): 태그 안에 표시되는 실제 내용

제목 태그

웹페이지의 제목과 부제목을 나타내는 태그입니다. h1부터 h6까지 있습니다.

<h1>가장 큰 제목</h1>

단락 태그

텍스트 단락을 정의합니다. 대부분의 텍스트 내용은 p 태그로 묶습니다.

<p>단락 내용입니다</p>

링크 태그

다른 웹페이지로 이동할 수 있는 하이퍼링크를 생성합니다.

<a href="URL">링크 텍스트</a>

이미지 태그

웹페이지에 이미지를 삽입합니다. 닫는 태그가 없는 빈 요소입니다.

<img src="이미지경로" alt="대체텍스트">

목록 태그

순서가 있는(ol) 또는 순서가 없는(ul) 목록을 만듭니다.

<ul><li>항목</li></ul>

테이블 태그

행과 열로 구성된 데이터 테이블을 생성합니다.

<table><tr><td>셀</td></tr></table>
"HTML은 웹페이지의 구조를 정의하는 마크업 언어로, 웹 자동화의 기초가 됩니다."
