PyAutoGUI를 활용한 인터넷 제어
화면 기반 자동화로 Selenium의 한계 극복하기
PyAutoGUI로 웹 브라우저를 제어하는 기본 사용법을 알려줘

PyAutoGUI 소개

PyAutoGUI는 마우스와 키보드를 프로그래밍 방식으로 제어하는 크로스 플랫폼 Python 모듈입니다. 화면 기반 자동화를 통해 Selenium으로 접근하기 어려운 환경(Flash, ActiveX 등)에서도 웹 브라우저 자동화가 가능합니다.

Python import pyautogui

# PyAutoGUI 안전 장치 설정 (커서를 화면 구석으로 이동하면 자동 종료)
pyautogui.FAILSAFE = True

# 마우스 이동 속도 설정 (0은 즉시, 1은 느리게)
pyautogui.PAUSE = 0.5

# 화면 해상도 확인
screen_width, screen_height = pyautogui.size()
print(f"화면 해상도: {screen_width}x{screen_height}")

마우스 제어

마우스 이동, 클릭, 드래그, 스크롤 등 다양한 마우스 작업을 자동화할 수 있습니다.

키보드 제어

키 입력, 단축키 실행, 문자열 타이핑 등의 키보드 작업을 자동화할 수 있습니다.

화면 인식

스크린샷을 찍고 이미지 인식을 통해 화면 요소를 찾을 수 있습니다.

창 관리

활성화된 창 정보 가져오기, 창 위치 변경 등의 작업이 가능합니다.

Python # 마우스 제어 예제
# 특정 위치로 마우스 이동 후 클릭
pyautogui.moveTo(100, 200, duration=0.5)
pyautogui.click()

# 현재 위치에서 더블 클릭
pyautogui.doubleClick()

# 마우스 드래그 (100,200에서 300,400으로)
pyautogui.dragTo(300, 400, duration=1)

# 마우스 스크롤
pyautogui.scroll(-10) # 아래로 스크롤
Python # 키보드 제어 예제
# 문자열 입력
pyautogui.write('Hello, world!', interval=0.1)

# 특수 키 입력
pyautogui.press('enter')

# 단축키 사용 (Ctrl+C)
pyautogui.hotkey('ctrl', 'c')

# 이미지 검색으로 요소 찾기
location = pyautogui.locateOnScreen('login_button.png', confidence=0.8)
if location:
    pyautogui.click(location)
