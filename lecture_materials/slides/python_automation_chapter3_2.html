판다스 데이터프레임 기초

업무 자동화의 핵심 도구인 판다스(Pandas) 데이터프레임의 기본 사용법을 알아봅니다.

데이터프레임이란? 핵심 도구

데이터프레임(DataFrame)은 판다스 라이브러리의 2차원 테이블 구조로, 행과 열로 구성되어 있습니다. 엑셀 시트와 유사하지만 더 강력한 데이터 조작 기능을 제공합니다.

  • 각 열은 서로 다른 데이터 타입을 가질 수 있습니다 (숫자, 문자열, 날짜 등)
  • 행과 열에 이름(라벨)을 지정할 수 있습니다
  • SQL, 엑셀과 같은 작업을 코드로 수행할 수 있습니다
  • 대용량 데이터 처리에 최적화되어 있습니다
데이터프레임 생성하기
직원 정보를 담은 간단한 판다스 데이터프레임 생성 코드를 작성해줘
import pandas as pd
# 딕셔너리로부터 데이터프레임 생성
employee_data = {
"이름": ["홍길동", "김철수", "이영희", "박지민"],
"부서": ["개발", "마케팅", "인사", "개발"],
"연봉": [5000, 4800, 5200, 5500],
"입사일": ["2020-01-15", "2019-05-20", "2021-03-10", "2018-12-05"]
}
df = pd.DataFrame(employee_data)
print(df)
외부 데이터 불러오기

실무에서는 주로 외부 파일에서 데이터를 불러와 데이터프레임을 생성합니다.

CSV, Excel, SQL 데이터를 판다스로 불러오는 코드 예제를 보여줘
# CSV 파일 불러오기
df_csv = pd.read_csv('데이터.csv', encoding='utf-8')
# Excel 파일 불러오기
df_excel = pd.read_excel('보고서.xlsx', sheet_name='Sheet1')
# SQL 데이터베이스에서 불러오기
import sqlite3
conn = sqlite3.connect('데이터베이스.db')
df_sql = pd.read_sql('SELECT * FROM 직원', conn)
