from openpyxl import Workbook from openpyxl.styles import Font, PatternFill # 새로운 워크북 생성 wb = Workbook() ws = wb.active ws.title = "직원 정보" # 헤더 추가 headers = ['이름', '부서', '직급', '입사일'] for col, header in enumerate(headers, 1): cell = ws.cell(row=1, column=col) cell.value = header cell.font = Font(bold=True) cell.fill = PatternFill('solid', fgColor='4B8BBE') # 데이터 추가 data = [ ['홍길동', '개발팀', '대리', '2022-01-01'], ['김영희', '마케팅팀', '과장', '2021-03-15'] ] for row, employee in enumerate(data, 2): for col, value in enumerate(employee, 1): ws.cell(row=row, column=col).value = value # 파일 저장 wb.save('직원명단.xlsx')
• Workbook(): 새로운 Excel 파일 생성
• ws.title: 워크시트 이름 설정
• Font(): 글꼴 스타일 설정 (굵게, 크기, 색상 등)
• PatternFill(): 셀 배경색 설정
• enumerate(): 인덱스와 값을 함께 순회
• wb.save(): 엑셀 파일 저장

주요 기능

  • 새로운 워크북 생성 및 기존 워크북 열기
  • 셀 값 읽기/쓰기 및 서식 지정
  • 여러 워크시트 관리
  • 수식 사용 및 계산
  • 차트 및 이미지 삽입
