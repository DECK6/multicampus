인덱싱과 슬라이싱 개념 설명
파이썬에서 인덱싱(Indexing)과 슬라이싱(Slicing)은 시퀀스 자료형(리스트, 문자열, 튜플 등)에서 요소에 접근하는 강력한 방법입니다. 이 기능을 이용하면 시퀀스의 특정 요소나 부분집합을 쉽게 추출할 수 있습니다.
인덱싱 (Indexing)

인덱싱은 시퀀스에서 특정 위치의 단일 요소를 추출하는 방법입니다.

P
Y
T
H
O
N
0
1
2
3
4
5
-6
-5
-4
-3
-2
-1
# 인덱싱 기본 구문
sequence[index]

# 예시
text = "PYTHON"
text[0]     # 'P' (첫 번째 문자)
text[2]     # 'T' (세 번째 문자)
text[-1]    # 'N' (마지막 문자)
text[-3]    # 'H' (뒤에서 세 번째 문자)
인덱싱 주의사항
  • 인덱스는 0부터 시작합니다 (첫 번째 요소의 인덱스는 0)
  • 음수 인덱스는 뒤에서부터 접근합니다 (-1은 마지막 요소)
  • 범위를 벗어난 인덱스 접근은 IndexError를 발생시킵니다
슬라이싱 (Slicing)

슬라이싱은 시퀀스에서 요소들의 부분집합을 추출하는 방법입니다.

P
Y
T
H
O
N
0
1
2
3
4
5
# 슬라이싱 기본 구문
sequence[start:stop:step]

# 예시
text = "PYTHON"
text[1:4]     # 'YTH' (인덱스 1부터 3까지)
text[:3]      # 'PYT' (처음부터 인덱스 2까지)
text[3:]      # 'HON' (인덱스 3부터 끝까지)
text[::2]     # 'PTO' (처음부터 끝까지 2단계씩)
start
슬라이스 시작 인덱스
(생략 시 처음부터)
stop
슬라이스 종료 인덱스
(이 인덱스 미포함, 생략 시 끝까지)
step
인덱스 증가폭
(생략 시 1, 음수면 역방향)

인덱싱과 슬라이싱 활용 예제
# 인덱싱과 슬라이싱 활용 예제

# 문자열에서의 인덱싱과 슬라이싱
text = "Python Programming"

# 인덱싱 - 특정 위치의 문자 접근
first_char = text[0] # 'P'
sixth_char = text[5] # 'n'
last_char = text[-1] # 'g'
print(f"첫 글자: {first_char}, 여섯 번째 글자: {sixth_char}, 마지막 글자: {last_char}")

# 슬라이싱 - 부분 문자열 추출
first_word = text[:6] # 'Python'
second_word = text[7:] # 'Programming'
sub_string = text[3:10] # 'hon Pro'
print(f"첫 단어: {first_word}, 두 번째 단어: {second_word}")

# 스텝 사용하기
every_second = text[::2] # 'Pto rgamn'
reversed_text = text[::-1] # 'gnimmargorP nohtyP'
print(f"2칸 간격: {every_second}, 역순: {reversed_text}")

# 리스트에서의 인덱싱과 슬라이싱
numbers = [10, 20, 30, 40, 50, 60, 70, 80, 90]

# 인덱싱
first_item = numbers[0] # 10
middle_item = numbers[4] # 50
last_item = numbers[-1] # 90

# 슬라이싱
first_three = numbers[:3] # [10, 20, 30]
middle_three = numbers[3:6] # [40, 50, 60]
last_three = numbers[-3:] # [70, 80, 90]
print(f"처음 세 개: {first_three}, 중간 세 개: {middle_three}, 마지막 세 개: {last_three}")

# 슬라이싱으로 리스트 복사하기
copy_of_numbers = numbers[:] # 전체 리스트의 복사본 생성

# 슬라이싱으로 리스트 수정하기
numbers[2:5] = [35, 45, 55] # 인덱스 2, 3, 4의 요소를 한 번에 변경
print(f"수정된 리스트: {numbers}")
