BeautifulSoup Requests Selenium Scrapy

Pandas NumPy Matplotlib Openpyxl

Pillow (PIL) ReportLab PyPDF2 FPDF

SMTPLIB win32com Twilio Telegram API

PyAutoGUI PyGetWindow keyboard mouse
