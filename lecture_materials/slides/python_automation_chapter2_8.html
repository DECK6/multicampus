숫자형과 문자열 자료형
숫자형 (Numeric)
숫자형은 수치 데이터를 저장하는 자료형입니다. 파이썬에서는 주로 정수형(int)과 실수형(float)을 사용합니다.
정수형(int): 소수점이 없는 숫자 (예: 1, 42, -7)
실수형(float): 소수점이 있는 숫자 (예: 3.14, -0.001, 2.0)
# 숫자형 예제 a = 10 # 정수형 b = 3.14 # 실수형 c = 2e3 # 지수 표기법 (2000.0) d = 0xFF # 16진수 (255) e = 0o10 # 8진수 (8) f = 0b1010 # 2진수 (10)
숫자형 연산: 숫자형에는 다양한 연산자를 사용할 수 있습니다.
+, -, *, / 기본 사칙연산
**, // 제곱과 몫 연산
% 나머지 연산
round(), abs() 반올림, 절대값 함수
문자열 (String)
문자열은 텍스트 데이터를 저장하는 자료형입니다. 작은따옴표(') 또는 큰따옴표(")로 감싸서 표현합니다.
# 문자열 생성 str1 = '작은따옴표 문자열' str2 = "큰따옴표 문자열" str3 = '''여러 줄 문자열''' str4 = """큰따옴표 여러 줄"""
문자열 연산과 메서드: 문자열은 다양한 연산과 메서드를 지원합니다.
연결: + 연산자로 문자열을 이어붙일 수 있음
반복: * 연산자로 문자열을 반복할 수 있음
upper() 모든 문자를 대문자로 변환
lower() 모든 문자를 소문자로 변환
strip() 문자열 양쪽의 공백 제거
replace() 특정 문자를 다른 문자로 대체
split() 문자열을 특정 구분자로 나누기

숫자형과 문자열 활용 예제
# 숫자형과 문자열 자료형 활용 예제 # 숫자형 활용 num1 = 10 num2 = 3.5 # 기본 연산 sum_result = num1 + num2 # 13.5 product = num1 * num2 # 35.0 power = num1 ** 2 # 100 (10의 제곱) remainder = num1 % 3 # 1 (10을 3으로 나눈 나머지) # 내장 함수 활용 rounded = round(num2, 0) # 4.0 (반올림) absolute = abs(-num1) # 10 (절대값) # 문자열 활용 text1 = "Python" text2 = '프로그래밍' # 문자열 연산 combined = text1 + " " + text2 # "Python 프로그래밍" repeated = text1 * 3 # "PythonPythonPython" # 문자열 메서드 활용 upper_case = text1.upper() # "PYTHON" replaced = text1.replace('P', 'J') # "Jython" # 문자열 포매팅 age = 25 message = f"{text1} 언어는 {age}살 이상이면 누구나 배울 수 있습니다." print(message) # "Python 언어는 25살 이상이면 누구나 배울 수 있습니다." # 문자열 분리와 결합 sentence = "파이썬,자바,C++,자바스크립트" languages = sentence.split(',') # ['파이썬', '자바', 'C++', '자바스크립트'] joined = " - ".join(languages) # "파이썬 - 자바 - C++ - 자바스크립트"
