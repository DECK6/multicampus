실습 과제

기본 과제

주어진 코드를 활용하여 다음 작업을 수행하세요:

  • 네이버 뉴스에서 "파이썬" 키워드로 검색하기
  • 검색 결과를 엑셀 파일로 저장하기
  • 워드클라우드 생성하여 PPT에 삽입하기
  • 결과물을 이메일로 전송하기
keyword = "파이썬" email = "[email protected]" main(keyword, email)

심화 과제

프로그램을 다음과 같이 개선해보세요:

  • 여러 키워드를 동시에 검색하도록 수정
  • 검색 결과를 데이터베이스에 저장
  • 감정 분석을 추가하여 긍정/부정 뉴스 분류
  • 매일 정해진 시간에 자동으로 실행되도록 설정
keywords = ["파이썬", "인공지능", "자동화"] schedule.every().day.at("09:00").do(batch_job)
