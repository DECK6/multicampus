리스트 생성과 요소 접근 방법
리스트 생성 방법
1. 리터럴 표기법 (대괄호 사용)

가장 일반적인 방법으로, 대괄호 [ ] 안에 요소를 쉼표로 구분하여 작성합니다.

fruits = ["사과", "바나나", "오렌지"]
empty_list = []
2. list() 생성자 사용

다른 이터러블(iterable) 객체로부터 리스트를 생성할 때 유용합니다.

numbers = list(range(1, 6))  # [1, 2, 3, 4, 5]
chars = list("Python")     # ['P', 'y', 't', 'h', 'o', 'n']
empty = list()            # []
3. 리스트 컴프리헨션(List Comprehension)

간결하고 효율적인 방법으로 기존 리스트를 기반으로 새 리스트를 생성합니다.

squares = [x**2 for x in range(1, 6)]  # [1, 4, 9, 16, 25]
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]
중첩 리스트 (Nested Lists)

리스트는 다른 리스트를 요소로 포함할 수 있으며, 이를 중첩 리스트 또는 다차원 리스트라고 합니다.

matrix = [
    [1, 2, 3],  # 첫 번째 행
    [4, 5, 6],  # 두 번째 행
    [7, 8, 9]   # 세 번째 행
]

중첩 리스트 요소 접근:

first_row = matrix[0]       # [1, 2, 3]
element = matrix[1][2]    # 두 번째 행, 세 번째 열: 6

파이썬에서는 인덱스를 사용하여 리스트의 개별 요소에 접근할 수 있습니다. 인덱스는 0부터 시작하며, 음수 인덱스를 사용하면 뒤에서부터 접근할 수 있습니다.

letters = ['A', 'B', 'C', 'D', 'E']

# 양수 인덱스 접근
first = letters[0]    # 'A'
third = letters[2]    # 'C'

# 음수 인덱스 접근
last = letters[-1]    # 'E'
second_last = letters[-2]  # 'D'
요소 수정: 인덱스를 사용하여 특정 위치의 요소 값을 변경할 수 있습니다. 
letters[1] = 'X'  # ['A', 'X', 'C', 'D', 'E']
인덱스 오류: 존재하지 않는 인덱스에 접근하면 IndexError가 발생합니다. 
letters[10]  # IndexError: list index out of range

리스트 생성과 요소 접근 예제
# 다양한 방법으로 리스트 생성하기

# 1. 리터럴 표기법으로 생성
colors = ["빨강", "파랑", "초록", "노랑"]
print(f"색상 리스트: {colors}") # 출력: 색상 리스트: ['빨강', '파랑', '초록', '노랑']

# 2. list() 생성자로 생성
numbers = list(range(1, 6))
print(f"숫자 리스트: {numbers}") # 출력: 숫자 리스트: [1, 2, 3, 4, 5]

characters = list("Python")
print(f"문자 리스트: {characters}") # 출력: 문자 리스트: ['P', 'y', 't', 'h', 'o', 'n']

# 3. 리스트 컴프리헨션으로 생성
squares = [x**2 for x in range(1, 6)]
print(f"제곱 리스트: {squares}") # 출력: 제곱 리스트: [1, 4, 9, 16, 25]

even_numbers = [x for x in range(10) if x % 2 == 0]
print(f"짝수 리스트: {even_numbers}") # 출력: 짝수 리스트: [0, 2, 4, 6, 8]

# 4. 중첩 리스트 생성
matrix = [
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]
]
print(f"행렬: {matrix}")

# 인덱스로 요소 접근하기
first_color = colors[0]
last_color = colors[-1]
print(f"첫 번째 색: {first_color}, 마지막 색: {last_color}")

# 중첩 리스트 요소 접근
center_element = matrix[1][1] # 가운데 요소
print(f"행렬의 중앙 요소: {center_element}") # 출력: 행렬의 중앙 요소: 5

# 요소 수정하기
colors[2] = "보라" # '초록'을 '보라'로 변경
print(f"수정된 색상 리스트: {colors}")

# 다양한 자료형이 포함된 리스트
mixed = [1, "안녕", 3.14, True, [1, 2]]
print(f"혼합 리스트: {mixed}")
print(f"혼합 리스트의 세 번째 요소: {mixed[2]}") # 출력: 혼합 리스트의 세 번째 요소: 3.14
