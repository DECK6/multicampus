Gmail SMTP 설정 가이드

Gmail SMTP를 사용하기 위한 필수 설정 단계를 알아보겠습니다.

Step 1

2단계 인증 활성화

1. Google 계정 관리로 이동
2. 보안 탭 선택
3. 2단계 인증 찾기 및 '시작하기' 클릭
4. 화면의 지시에 따라 설정 완료

Step 2

앱 비밀번호 생성

1. Google 계정 보안 설정으로 이동
2. '앱 비밀번호' 선택
3. 앱 선택: '메일'
4. 기기 선택: '기타'
5. 생성된 16자리 비밀번호 저장

Step 3

SMTP 서버 정보

• SMTP 서버: smtp.gmail.com
• 포트: 587 (TLS) / 465 (SSL)
• 사용자 이름: 본인의 Gmail 주소
• 비밀번호: 생성된 앱 비밀번호

Step 4

보안 설정 확인

• 보안 수준이 낮은 앱 액세스 허용
• 방화벽 설정 확인
• SSL/TLS 프로토콜 지원 확인
• 네트워크 제한사항 확인

주의사항

앱 비밀번호는 절대 공유하지 마시고, 안전하게 보관하세요. 비밀번호가 노출되었다면 즉시 재생성이 필요합니다. 또한, 회사 정책에 따라 일부 설정이 제한될 수 있으니 IT 부서와 확인이 필요할 수 있습니다.
