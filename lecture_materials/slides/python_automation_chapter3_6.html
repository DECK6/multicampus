반복문 - for 루프

반복문은 동일한 코드를 여러 번 실행하여 반복 작업을 자동화하는 핵심 구문입니다. for 루프는 가장 많이 사용되는 반복문입니다.

기본 for 루프

파이썬의 for 루프는 시퀀스(리스트, 튜플, 문자열 등)의 각 항목을 순회하면서 코드 블록을 실행합니다.

시작
시퀀스의 다음 항목
코드 블록 실행
모든 항목 순회 완료?
종료
파이썬의 기본적인 for 루프 예제 코드를 보여줘
# 리스트를 순회하는 기본 for 루프
과일_목록 = ['사과', '바나나', '체리', '딸기']
for 과일 in 과일_목록:
print(f"과일: {과일}")
# range()를 사용한 숫자 범위 순회
for i in range(5): # 0, 1, 2, 3, 4
print(f"숫자: {i}")
출력 결과:
과일: 사과
과일: 바나나
과일: 체리
과일: 딸기
숫자: 0
숫자: 1
...(생략)
인덱스와 값 함께 순회하기

enumerate() 함수를 사용하면 시퀀스의 인덱스와 값을 동시에 순회할 수 있습니다.

enumerate를 사용해서 인덱스와 값을 함께 출력하는 예제를 보여줘
과일_목록 = ['사과', '바나나', '체리']
# enumerate() 사용하기
for 인덱스, 과일 in enumerate(과일_목록):
print(f"{인덱스}번째: {과일}")
# 시작 인덱스 변경
for 인덱스, 과일 in enumerate(과일_목록, start=1):
print(f"{인덱스}번째: {과일}")
출력 결과:
0번째: 사과
1번째: 바나나
2번째: 체리
1번째: 사과
2번째: 바나나
3번째: 체리
실무 활용 Tip

엑셀 시트의 행 번호를 표시하거나, 데이터 처리 시 진행 상황을 표시할 때 enumerate()가 매우 유용합니다.

딕셔너리 순회하기

딕셔너리는 키, 값, 또는 키-값 쌍을 순회할 수 있습니다.

딕셔너리를 순회하는 다양한 방법을 보여주는 코드 예시
직원_정보 = {
'홍길동': '개발팀',
'김철수': '마케팅팀',
'이영희': '인사팀'
}
# 키만 순회
for 이름 in 직원_정보.keys():
print(f"직원 이름: {이름}")
# 값만 순회
for 부서 in 직원_정보.values():
print(f"부서: {부서}")
# 키-값 쌍 순회
for 이름, 부서 in 직원_정보.items():
print(f"{이름}은(는) {부서}에 소속되어 있습니다.")
items() 메서드 출력 결과:
홍길동은(는) 개발팀에 소속되어 있습니다.
김철수은(는) 마케팅팀에 소속되어 있습니다.
이영희은(는) 인사팀에 소속되어 있습니다.
데이터프레임 순회하기

판다스 데이터프레임을 다양한 방식으로 순회하여 데이터를 처리할 수 있습니다.

판다스 데이터프레임을 순회하는 다양한 방법을 보여줘
import pandas as pd
df = pd.DataFrame({
'이름': ['홍길동', '김철수', '이영희'],
'나이': [25, 30, 28],
'급여': [3000, 3500, 3200]
})
# 행별 순회 (iterrows)
for index, row in df.iterrows():
print(f"행 {index}: {row['이름']}의 급여는 {row['급여']}입니다.")
# apply 함수로 행 처리
def 급여_인상(row):
return row['급여'] * 1.1
df['인상된_급여'] = df.apply(급여_인상, axis=1)
효율적인 데이터프레임 처리

데이터프레임을 순회할 때는 iterrows()보다 apply() 또는 벡터화 연산을 사용하는 것이 더 효율적입니다. 대량의 데이터를 처리할 때는 특히 성능 차이가 큽니다.
