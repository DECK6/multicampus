파이썬 함수의 이해
함수란 무엇인가?

함수는 특정 작업을 수행하는 코드 블록으로, 코드의 재사용성을 높이고 프로그램의 구조화를 도와줍니다. 파이썬에서 함수는 def 키워드를 사용하여 정의합니다.

함수는 입력값(매개변수)을 받아 처리한 후, 결과값을 반환할 수 있습니다. 함수를 사용하면 동일한 코드를 반복해서 작성할 필요가 없어 코드의 가독성과 유지보수성이 향상됩니다.

함수의 기본 구조 
def 함수명(매개변수1, 매개변수2, ...):
    """함수 설명 (독스트링)"""
    # 함수 본문
    # 코드 블록
    
    return 반환값  # 선택사항
함수의 종류
내장 함수
파이썬에 기본적으로 포함된 함수들입니다. print(), len(), type(), range() 등이 있습니다.
사용자 정의 함수
사용자가 직접 정의한 함수로, 특정 업무나 작업에 맞게 커스터마이징된 기능을 제공합니다.
람다 함수(익명 함수)
이름 없이 한 줄로 정의되는 간단한 함수입니다. 주로 filter(), map() 등과 함께 사용됩니다.
재귀 함수
자기 자신을 호출하는 함수로, 복잡한 문제를 간단한 하위 문제로 나누어 해결할 때 유용합니다.

# 1. 기본 함수 예제 - 엑셀 파일에서 데이터 추출
def extract_data_from_excel(file_path, sheet_name=0):
    """
    엑셀 파일에서 데이터를 추출하는 함수
    
    Args:
        file_path (str): 엑셀 파일 경로
        sheet_name (str or int): 시트 이름 또는 인덱스
        
    Returns:
        pandas.DataFrame: 추출된 데이터프레임
    """
    import pandas as pd
    
    try:
        df = pd.read_excel(file_path, sheet_name=sheet_name)
        print(f"{file_path}에서 {len(df)}개의 데이터를 추출했습니다.")
        return df
    except Exception as e:
        print(f"데이터 추출 중 오류 발생: {e}")
        return None

# 2. 람다 함수 예제 - 숫자 리스트 필터링
numbers = [10, 25, 3, 41, 8, 15, 22]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(f"짝수 목록: {even_numbers}")  # [10, 8, 22]
매개변수 유형
ChatGPT 프롬프트: "파이썬 함수의 다양한 매개변수 유형과 사용 예를 보여주세요." 
# 1. 기본 매개변수(Default Parameter)
def greet(name, message="안녕하세요"):
    return f"{name}님, {message}!"

# 2. 가변 인자(*args)
def calculate_sum(*numbers):
    return sum(numbers)

# 3. 키워드 가변 인자(**kwargs)
def create_profile(**user_info):
    profile = ""
    for key, value in user_info.items():
        profile += f"{key}: {value}\n"
    return profile

# 사용 예시
print(greet("홍길동"))                         # 홍길동님, 안녕하세요!
print(greet("김철수", "반갑습니다"))            # 김철수님, 반갑습니다!
print(calculate_sum(1, 2, 3, 4, 5))           # 15
print(create_profile(이름="이영희", 나이=30, 직업="개발자"))
