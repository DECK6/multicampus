파이썬 기본연산 - 비교/논리연산자
파이썬에서 비교 연산자와 논리 연산자는 조건을 판단하고 결과에 따른 코드 실행을 제어하는 데 필수적입니다. 이 연산자들은 데이터 검증, 조건부 로직 구현, 반복문 제어 등 다양한 상황에서 활용됩니다.
비교 연산자 (Comparison Operators)
연산자 설명 예시
== 같음 x == y
!= 같지 않음 x != y
> 초과 x > y
< 미만 x < y
>= 이상 x >= y
<= 이하 x <= y
논리 연산자 (Logical Operators)
연산자 설명 예시
and 두 조건이 모두 참일 때 참 x > 0 and x < 10
or 두 조건 중 하나라도 참이면 참 x < 0 or x > 10
not 조건의 결과를 반대로 변환 not x > 10

참고: 파이썬에서는 숫자 0, 빈 문자열(""), 빈 리스트([]), None 등은 False로 평가되고, 그 외의 값들은 True로 평가됩니다.

비교 및 논리 연산자 활용 예제
# 비교 연산자와 논리 연산자 활용 예제 # 나이와 회원 상태 변수 설정 age = 25 is_member = True points = 750 # 비교 연산자 사용 is_adult = age >= 18 print(f"성인 여부: {is_adult}") # 출력: 성인 여부: True # 논리 연산자 사용 if age >= 18 and is_member: print("성인 회원입니다.") # 출력: 성인 회원입니다. # 복합 조건문 if age >= 18 and (is_member or points > 500): discount = 20 elif age >= 18 or points > 1000: discount = 10 else: discount = 0 print(f"적용 할인율: {discount}%") # 출력: 적용 할인율: 20% # not 연산자 사용 if not is_member: print("회원이 아닙니다.") else: print("회원입니다.") # 출력: 회원입니다. # 단락 평가(Short-circuit evaluation) 예시 result = is_member and "VIP" or "손님" print(f"고객 등급: {result}") # 출력: 고객 등급: VIP
