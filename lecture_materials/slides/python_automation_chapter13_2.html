환경 설정
Python 3.7 이상이 설치되어 있어야 하며, pip가 정상적으로 동작하는지 확인합니다. 가상환경 사용을 권장합니다.
2
라이브러리 임포트
설치된 라이브러리를 프로젝트에서 import하여 사용합니다. 각 라이브러리의 기능별 모듈을 필요에 따라 선택적으로 임포트할 수 있습니다.
3
버전 호환성
MS Office 2007 이상의 버전(.xlsx, .pptx, .docx)과 호환됩니다. 이전 버전 파일 형식은 지원되지 않으므로 주의가 필요합니다.
4
문제 해결
설치 중 오류 발생 시 pip를 최신 버전으로 업그레이드하거나, 필요한 시스템 종속성을 확인하세요. Windows 사용자는 Visual C++ Build Tools가 필요할 수 있습니다.
